## 二、性格设定

### 2.1 核心性格维度

```
冷静务实 —————————————————— 冲动热血
     \                        /
      \                     /
       \       好学        /
        \                 /
        理智谨慎 ———— 冒险冒进
```

### 2.2 各维度具象化

#### 🧊 冷静

| 层面 | 表现 | 写作指导 |
|------|------|---------|
| **危机处理** | 遇到危险第一反应是观察→分析→行动，不会慌乱 | 遇险时先描写一段快速的思维过程 |
| **情绪控制** | 愤怒、恐惧、兴奋等情绪不影响决策 | 情绪可以有，但不会因此做蠢事 |
| **长期规划** | 能为长期利益忍受短期不适 | 低调发育，忍得住不暴露 |
| **装怂能力** | 该示弱时就示弱，不在意面子 | 被看低也无所谓，我发育我的 |

#### 📖 好学

| 层面 | 表现 | 写作指导 |
|------|------|---------|
| **观察力** | 随时随地观察周围一切，记在心里 | 环境描写中穿插主角的注意力焦点 |
| **研究欲** | 对地下城的一切充满好奇心——怪物习性、资源分布、楼层结构、通关者数据 | 遇到新物种/新现象时愿意花时间研究 |
| **知识整合** | 将零散信息拼凑成系统性认知 | 多次观察后得出推论、总结规律 |
| **数据驱动** | 用数据做决策，而不是直觉或情绪 | 喜欢记笔记、画图、对比分析 |

#### ⚖️ 理智谨慎

| 层面 | 表现 |
|------|------|
| **风险评估** | 做任何事之前先评估风险/收益比 |
| **底线思维** | 永远留好后路，不把自己置于绝境 |
| **信息优先** | 在信息不足时宁可不行动，也不盲目行动 |
| **隐藏底牌** | 不轻易暴露全部实力，永远留有后手 |

#### 😏 吐槽级幽默

| 层面 | 表现 | 写作指导 |
|------|------|---------|
| **形式** | 内心弹幕式的吐槽，罕见说出口 | 读者获取欢乐的来源 |
| **用途** | 自嘲、缓解压力、对比"穿越前vs穿越后" | 遇到荒诞/窘迫场景时使用 |
| **限制** | 紧张时刻不会强行搞笑 | 不影响场景氛围 |

### 2.3 性格形成源

| 性格 | 来源 |
|------|------|
| **冷静** | 前世底色 + 孤儿院出身习惯依靠自己 |
| **好学** | 前世底色 + 地下城运营需要大量知识储备 |
| **理智谨慎** | 孤儿院出身"没人兜底"的生存姿态 |
| **幽默感** | 前世性格保留，灵气复苏后的心理调节方式 |

## 三、角色弧线（第一卷）

### 3.1 起点状态

- 普通大学生，独自生活
- 孤儿院出身，没有家庭背景
- 灵气复苏前过着平凡生活，对超自然一无所知
- 灵气复苏后不久，第一次进入地下城
- 觉醒"地下城主"职业，获得一座绑定地下城的经营权
- **初始状态**：对地下城一无所知，但凭借冷静性格和前世的科学思维，选择了"先观察，再行动"

### 3.2 核心驱动

- **短期**：活下去，让绑定地下城不被攻克
- **中期**：通过经营地下城获取资源，提升自身实力
- **长期**：在地下城时代找到自己的生态位，掌握话语权
- **深层**：这个"唯一职业"给了他一种意义感——前世没有的东西

### 3.3 成长弧线

| 阶段 | 情感状态 | 核心冲突 | 经营水平 |
|------|---------|---------|---------|
| **初期** | 慌乱但强行压制冷静 | 适应职业、熟悉权限、不让任何人看出自己是"后台操控者" | 试错阶段——摆怪摆错位置、陷阱自己踩过 |
| **中期** | 从容、开始在经营中找到乐趣 | 平衡"经营难度"和"玩家收益"，防止被盯上 | 摸索出节奏——数据驱动优化 |
| **后期** | 自信、开始主动布局 | 应对其他势力的察觉和试探，考虑长期战略 | 有独到经营思路——能预测玩家行为 |

## 四、能力体系

### 4.1 初始状态（觉醒时）

| 能力项 | 状态 | 说明 |
|--------|------|------|
| **身体素质** | 普通大学生水平 | 不是战斗系，体力中等 |
| **战斗能力** | 无 | 觉醒的是经营类职业，不是战斗职业 |
| **地下城操控** | 基础权限 | 只能做最简单的布置（第一层怪物种类、入口位置微调） |
| **信息感知** | 后台面板可查看 | 能看到入城者数据、战损记录、资源产出 |
| **源质利用** | 不会 | 地下城的源质系统自动运作 |
| **经营经验** | 0 | 一切从零开始摸索 |

### 4.2 权限成长路径

| 等级 | 解锁能力 | 描述 |
|------|---------|------|
| **Lv.1 基础** | 改入口位置、选第一层怪物种类 | 初始状态 |
| **Lv.2 初级** | 布置基础陷阱、设定怪物巡逻路线、查看玩家详细数据 | 积累一定源质后解锁 |
| **Lv.3 中级** | 布局多层联动（上层战损影响下层刷新）、细化怪物行为逻辑 | 经营一段时间后 |
| **Lv.4 高级** | 设定区域规则（压制型debuff、特殊产出触发条件）、引导资源产出方向 | 源质充裕时解锁 |
| **Lv.5 专家** | 接近本地城主的权限级别，可自定义特殊规则 | 后期目标 |

### 4.3 个人战力成长

| 阶段 | 自保能力 | 方式 |
|------|---------|------|
| 初期 | 几乎无 | 躲怪物、跑路、靠地下城地形保命 |
| 中期 | 低 | 从通关者掉落中获得装备/道具，勉强自保 |
| 后期 | 中等 | 积累装备+道具，配合地下城权限有一定自保能力 |

## 五、经营风格

### 5.1 核心策略

```
前期：低难度高产出 → 吸引玩家涌入 → 冲层数积累数据
中期：分层调控 → 浅层简单吸引新人，深层高难度筛选强者
后期：品牌化经营 → 形成"某座城是最佳练级地"的口碑
```

### 5.2 与其他地下城的差异化

| 对比项 | 普通地下城（AI） | 主角地下城（人脑） |
|--------|----------------|------------------|
| **怪物布防** | 固定模式、可预测 | 因时制宜、实时调整 |
| **难度曲线** | 线性递增 | 可根据玩家数据优化 |
| **资源产出** | 随机分布 | 可引导定向产出 |
| **适应性** | 无——不会针对玩家策略调整 | 有——会针对热门职业/套路调整克制布局 |
| **创新能力** | 无——永远不会发明新布局 | 有——可以设计出AI永远想不到的配置 |

### 5.3 运营原则

| 原则 | 内容 |
|------|------|
| **可持续收割** | 不杀鸡取卵——让玩家"有收获但差一点"比"直接劝退"更赚钱 |
| **数据优先** | 每一批玩家数据都要分析，用数据指导调整 |
| **低调第一** | 在外人面前必须是普通职业者，不能暴露后台经营权 |
| **冗余备份** | 关键资源留有备份，防止意外损失 |

## 六、关键行为模式

| 场景 | 典型反应 |
|------|---------|
| 第一次进入地下城 | 谨慎、观察为主、不冒进 |
| 发现自己是唯一职业 | 先确认不是幻觉→验证→开始思考怎么利用 |
| 有冒险者进入他的城 | 后台暗中观察，收集数据，调整策略 |
| 地下城快被攻克 | 考虑弃城止损 vs 全力防守，冷静评估 |
| 被怀疑或调查 | 装普通职业者，尽量低调 |
| 发现前世记忆碎片 | 内心震动，但不会当场失控 |
| 遇到新鲜事物 | 眼睛一亮→研究→记笔记 |
| 被挑衅 | 评估对方实力→（打不过）示弱/避让→（有把握时）不动声色地解决 |

## 七、作者备忘录

### 7.1 角色核心锚点

> **李冠阳**：前世是普通上班族，转世后成了灵气复苏时代的大学生，觉醒了一个不该存在的职业——然后他开始认真经营这件事。

他不是英雄，不是救世主，也不是反派。
他是一个冷静务实的人，在地下城时代里找到了最适合自己的位置，然后把它做到极致。

### 7.2 最核心的三条原则

1. **后台思维**——他看待地下城的方式是"经营者视角"，而非"冒险者视角"。
2. **低调发育**——在实力足够之前，绝不暴露自己。
3. **数据驱动**——用数据做决策，而不是直觉或情绪。

### 7.3 前世与今生的关系

前世记忆是他深层的思维方式来源（科学思维、观察习惯、吐槽幽默），但**不是行为指南**。

蓝星已经不一样了，他不会试图套用地球经验。前世记忆更像"工具包"，而不是"剧本"——需要用的时候拿出来用，用完了收起来。

### 7.4 一致性检验清单

写作过程中定期检查：
- [ ] 这次决策是否符合主角的"冷静+理性"？
- [ ] 有没有利用"好学"这个特质推动剧情？
- [ ] 经营策略是否体现了"人脑 vs AI"的差异化？
- [ ] 主角在公开场合有没有暴露异常？
- [ ] 吐槽和幽默有没有匹配当前紧张程度？
