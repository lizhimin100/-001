# 2.3 神明体系

> 神明的本质、等级与分类——世界的上层力量架构。
> 核心设定：神明是极致理念的体现，每个神明都代表着一个深刻的思想或原则。
> 底层原理详见 `2.2_世界本质与构成.md`。

---

## 一、神明的本质

### 1.1 神明是什么

神明是**极致理念的化身**。

世界的本质由两部分构成——**思想理念**与**物质元素**。当一个理念被足够多的智慧生物相信、遵守、实践时，它就会凝聚成形，在世界底层中凝结出对应的力量。当这种凝聚达到临界点，神明的本质“神性”便诞生了。

这意味着：

- **神明不是造物主**——它们是世界底层理念的人格化体现或实践者
- **信仰塑造神明**——信徒的信仰会强化神明，也影响神明的倾向
- **理念先于神明**——"战斗"的理念存在，才有战斗之神
- **神明可以陨落**——当对应的理念被遗忘或被同等力量斩杀，神明也会消逝

### 1.2 神明与凡物的关系

| 维度 | 关系说明 |
|------|---------|
| 力量来源 | 神明的力量来自理念的凝聚和被信仰的规模 |
| 与凡物的互动 | 神明需要信徒来维持存在感，但部分超强神明对信仰的依赖较低 |
| 对世界的干预 | 神位越高，直接干预物质世界的限制越大 |
| 神职 | 每个神明掌管的神职范围决定其权柄边界 |

## 二、神明等级

| 神阶 | 说明 | 特征 | 战力对标 |
|------|------|------|---------|
| **圣者** | 神明在物质世界的化身/容器 | 拥有神明的一部分力量，可以被凡物杀死 | 传奇高段 |
| **半神** | 介于凡物与神明之间的存在 | 拥有神性血脉或部分权柄，可升格为真神 | 巅峰传奇 |
| **微弱神力** | 信仰者极少的真神 | 权柄范围极小，神力有限 | 传奇10-12段 |
| **弱小神力** | 有一定信仰规模的神明 | 掌管一个小领域，如河流之神、山丘之神 | 传奇13-15段 |
| **中等神力** | 主流文明常见的信仰对象 | 掌管一个或多个大领域，如战争之神、丰收之神 | 传奇16-18段 |
| **强大神力** | 广泛信仰的顶级神明 | 掌管核心领域，影响世界格局 | 超越传奇常规尺度 |
| **超强神力** | 超越普通神明范畴 | 接近世界底层的理念本身，如生命、死亡、时间 | 近乎概念化身 |
| **神上神** | 最上位的神明 | 近乎理念本身的化身，极少甚至从不干预 | 不可衡量 |

## 三、神明的诞生与陨落

### 3.1 诞生途径

| 途径 | 说明 | 例子 |
|------|------|------|
| **信仰凝聚** | 某个理念被广泛信仰，自然凝聚成神明 | 最常见的神明诞生方式 |
| **凡物升格** | 强大的凡物（通常为传奇强者）通过功绩或仪式升格为神 | 半神→真神的路径 |
| **神性继承** | 神明陨落后，其权柄被继承者获得 | 神战落败方的权柄被瓜分 |
| **概念自生** | 世界诞生之初即存在的古老神明 | 时间之神、命运之神等 |

### 3.2 陨落条件

- **信仰断绝**：信徒全部消失，神明随之消逝
- **理念消亡**：对应的原则不再被任何智慧生物理解或实践
- **神战陨落**：被更强大的神明击杀，权柄被掠夺
- **规则变迁**：世界底层规则发生变化，导致某些神明失去存在基础

## 四、神职体系

### 4.1 常见神职分类

| 神职大类 | 具体领域 | 通常神力等级 |
|---------|---------|------------|
| **自然** | 太阳、月亮、大地、海洋、森林、风暴、丰收 | 中等—强大 |
| **生命** | 生命、死亡、生育、医疗、瘟疫 | 中等—超强 |
| **文明** | 知识、工艺、法律、商业、旅行、艺术 | 中等—强大 |
| **战争** | 战争、战斗、勇气、征服、毁灭 | 弱等—强大 |
| **理念** | 正义、自由、秩序、混沌、命运、真理 | 不定（取决于广泛程度） |
| **元素** | 火、水、土、风、自然之灵 | 微—中等 |
| **憎恶** | 堕落、腐化、憎恨、痛苦、疯狂 | 特殊（憎恶之路） |

### 4.2 神职的排他性

- **同一神职不能有两个同等级神明**——除非共享权柄（双子神、神系内的分工）
- **神职可以重叠**——不同神系的神明可以掌管相同领域，但"本土地位"更高
- **神职可以被分割**——一个强大神力的陨落可能导致其权柄被多个较弱神明瓜分

## 五、憎恶之路

### 5.1 憎恶之神

除了正神体系，还存在**憎恶之神**——由毁灭性、扭曲性的极端理念凝聚而成。它们不依赖传统信仰，而是从恐惧、痛苦、堕落中汲取力量。

| 类别 | 说明 |
|------|------|
| **憎恶职业** | 通过扭曲、禁忌的手段强行改变自身职业路径，获得力量的同时承受诅咒 |
| **憎恶种族** | 通过禁忌仪式或堕落，整个种族发生异变，化为憎恶的存在 |

### 5.2 憎恶之路的特点

- 力量远超同等阶位的正常存在
- 代价极大：失去理智、被世界排斥、永世不得寸进
- 部分憎恶之神比正神更强大——因为"恐惧"和"痛苦"从不缺少信徒

## 六、神明与世界的关系

### 6.1 神国

每个真神都拥有自己的**神国**——位于物质世界之外的独立空间，是神明与其信徒灵魂的居所。神国存在于世界夹层或底层规则之中。

### 6.2 神明的限制

- **神位越高，直接干预物质世界的限制越大**
  - 微弱神力可以降临化身
  - 中等神力以上需要圣者作为容器
  - 超强神力几乎无法直接出现在物质世界
- **神明不能违背自身神职**——正义之神无法行不义之事（至少不能公开做）
- **神明可以被"弑杀"**——足够强大的凡物（传奇巅峰+天命之路）可以弑神

### 6.3 神明的分布（世界观应用参考）

在《龙文》的故事中，不同地区对神明的信仰各有侧重：

| 区域/阵营 | 主要信仰 |
|---------|---------|
| 人类帝国 | 秩序、正义、丰收、战争、商业等文明神 |
| 精灵王国 | 自然、生命、月亮、知识等自然神 |
| 矮人要塞 | 锻造、大地、工艺等实用领域 |
| 龙族领地 | 力量、智慧、财富、毁灭——龙族普遍崇拜自身的力量而非外神 |
| 兽人部落 | 战争、征服、力量、鲜血等暴力倾向 |
| 黑暗地域 | 憎恶之神、堕落、痛苦、暗影 |

## 七、写作运用指南

- **神明的存在感**：在故事前期，神明以"传说""信仰""祈祷"的形态出现，不必直接登场
- **神明的不可知**：凡人对神明的理解永远是片面的，不同文明对同一神明的认知可能截然不同
- **神话是武器**：每个神系都有自己的起源神话，这些神话本身就是世界观的一部分
- **神明不是NPC**：神明不宜太过频繁地直接与主角互动，保持神秘感和崇高感
- **弑神是里程碑**：当主角能弑神时，说明他已经站在世界之巅
