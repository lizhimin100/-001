# 03 地下城设定（地下城——超凡出现的主产物，职业者的试炼场与资源来源。）

地下城是**活体秘境**——在异空间混沌中生成的独立空间结构。
和所有职业者的绑定。也独立于任何单一个体。
>>>>注：普通地下城的核心意识不是AI，而是混沌孕育的懵懂新生意识（有本能智慧但无经验阅历）。主角的地下城核心分身则是穿越者意识——两者导致了行动和策略上有了本质区别。

## 一、地下城是什么

### 1.1 定义

地下城是**活体秘境**——在异空间混沌中生成的独立空间结构。每座地下城由若干层构成，每层一个独立空间，越深层的怪物越强、奖励越丰厚。

地下城不是"副本"——不是系统生成的临时关卡，而是**真实存在的异空间结构**，被攻克后永久消失，不会重置刷新。

### 1.2 地下城与世界的关系

异空间混沌（孕育期）→ 混沌停止、层数固定 → 降临蓝星（固定在某一地理位置）→ 职业者进入挑战 → 被攻克或持续存在

- 地下城既是超凡出现的**原因之一**，也是**产物**——降临带来的超凡能量扩散加速了全球的觉醒进程。
- 生成于异空间混沌中，混沌中孕育的"种子"决定了城的类型和基础属性。
- 降临后固定在蓝星某处，不可移动。

### 1.3 地下城与觉醒

规模最小的地下城（6层起，最浅层超凡能量稀薄）在发现觉醒规律的觉醒者之间俗称"**觉醒坑**"——是普通人晋升职业者的关键入口。

- 这类小型地下城释放的微量超凡能量能刺激普通人潜藏的职业资质，使部分人在城内战斗时当场觉醒。
- 觉醒后立刻获得**一转1级**，但由于已经身在城中，需要杀穿当前层才能出去。
- 因此小型地下城的实际死亡率不低——觉醒前的普通人太过脆弱，战斗中的任何失误都可能导致死亡。

> 当前时代，政府没有能力管控这些小型地下城。各处处于无人管理的自发状态。知道小型地下城能觉醒的人还不多，进入者主要靠运气和胆量。死亡率完全取决于个人。

## 二、生成与降临

> 当前时代背景：超凡出现初期（约3周内）。地下城刚刚开始降临——既是导致超凡出现的源头，也是超凡现象的集中体现。没有监控体系，没有统一应对方案。信息的传播速度远慢于地下城的出现速度。

### 2.1 混沌孕育期

每座地下城在异空间的混沌中孕育。此阶段：

- 层数持续增长——但城的"自觉成熟度"决定了何时自动现世。最低需发育到6层才会诞生意识——低于6层的城会在混沌中继续发育，不会主动现世。
  - **越早降临的城普遍层数越少（发育时间短），越晚降临的城层数越多。** 最早降临的城可能仅6-10层（小型），之后降临的城层数逐渐增多。
  - 6层是最低门槛——低于6层的城没有意识，不会现世。
- 混沌中孕育的类型决定城的分类（战斗型/资源型/迷宫型等）。
- 孕育期长短不一，从数月到数十年都有。
- 孕育期间外界无法探查，只有少数觉醒者能隐约感知到异空间的波动。

### 2.2 自动现世与定型降临

核心诞生意识/意识苏醒到一定程度 → 自动开启入口 → 现世。不是"选择降临"，而是**控制不住自动现世**。

#### 2.2.1 不稳定期（首降前——入口投影阶段）

地下城现世前，在现世降临处会有入口投影，此阶段：

- **投影而非实体入口：** 不是石门/光门等物理结构，而是空间褶皱——像是空气里被撕开一道不稳定的裂口，时而清晰时而模糊。投影肉眼可见，但不可触摸（手穿过去不会有触感）。
- **区域置换：** 投影本身不"吸人"。当生物进入投影一定范围内，会被地下城核心的溢出能量识别为"可摄入对象"——脚下/周围空间瞬间置换，人被转移到地下城内部。没有风压，没有拉扯感。
- **触发条件：** 投影不主动感知，不追踪生物。置换是核心能量溢出导致的**被动现象**——只有当生物恰好进入投影的混沌范围，且核心的不稳定能量刚好处于"溢出峰值"时，空间置换才会发生。不是投影在"吸人"，是核心在不稳定的状态下无法控制自己的能量边界。
- **此阶段是临时的：** 核心意识一旦发育完成（定型降临），投影消失，被永久性实体入口取代。

> 第1章入口投影即为此机制——投影的第一次显形，也是城即将降临的前兆。

#### 2.2.2 定型降临

入口**无法关闭**。一旦现世入口永久存在，固定在该地理位置。

普通地下城核心的"感觉发育好了"阈值不是固定的——有的城6层就觉得够，有的城12+层才觉得够。但最低阈值是6层——**低于6层的城没有一座会诞生意识。**

当混沌停止、层数固定时，地下城完成定型降临：
- 降临位置不可预测——可能出现在荒野、城市地下、海洋、天空。
- 降临时会引发波动，强力的城甚至有天象异变。
- 降临后层数**不再变化**，位置**不可移动**。
- **当前没有系统化的降临监测机制**——各国政府还不知道如何系统应对，大多数地下城降临是被普通人目击发现的。

### 2.3 降临类型

| 类型 | 说明 | 应对方式 |
|------|------|---------|
| **自然降临** | 孕育完成后自发降临 | 最常见，目前没有系统监测手段 |
| **诱导降临** | 通过特殊手段提前触发 | 极稀有，当前时代无人知道此技术 |
| **连锁降临** | 某城被攻克后，附近混沌中相邻的城被触发降临 | 尚未有人类攻克过地下城，此情况尚未出现 |

### 2.4 应对现状

当前社会对地下城降临没有统一的应对策略：

- **被发现的地下城：** 谁先发现谁先探索。地方政府如果有觉醒者资源，会尝试组织；如果没有，就放着不管。
- **未发现的地下城：** 可能降临在无人区数日甚至数周后才被发现。
- **城市内降临：** 引发恐慌。地方政府紧急封锁现场，但不知道怎么处理——职业者不够、信息为零。
- **无人区降临：** 可能长时间无人发现。这种城会在无人知晓的情况下持续存在。

## 三、层数结构

### 3.1 基本规则

地下城由多层构成。每层一个独立空间，空间大小不一。

| 属性 | 说明 |
|------|------|
| **层数范围** | 6层+，按强度分为：小型6-20层，中型21-50层，大型51-90层，超大型91层以上 |
| **层间连接** | 层与层之间通过传送阵或通道相连 |
| **层内空间** | 少则足球场大小，多则整座城市规模 |
| **越层机制** | 部分特殊道具允许跳过中间层（极稀有） |

### 3.2 层数分布规律

不同规格的地下城，难度分布逻辑不同：

- **小型城（6-20层）**：整体难度适合一转及以下。从第1层到最后一层是线性爬坡，但全部停留在同一转位区间内。不存在"浅层适合一转、中层适合二转"的跨转位划分——小型城跨度不足以跨越转位门槛。
- **中型城（21-50层）**：前10-15层适合一转，中层开始逐步转向二转难度。
- **大型及以上（51层+）**：难度曲线跨越至少两个转位区间，前30%适合一转，中段适合二转及以上，深层可达三转甚至四转。

> 简言之：层的绝对编号不直接决定难度档次。城的规格决定了跨度，跨度决定了难度曲线能跨越几个转位。

### 3.3 层数命名规范

各层通常按顺序编号（第1层、第2层……）。在大中型城市中，职业者之间约定俗成：
- 1-5层称为"表层"（受表层环境机制影响，怪物强度略低于同城其他层）
- 6-15层称为"中层"
- 16层+称为"深层"
- 最后一层称为"核心层"

小型城（6-20层）的层数命名不严格适用以上约定，职业者通常直接以编号称呼。

## 四、地下城的构成要素

### 4.1 核心

每座地下城都有一个**核心**——位于最深处的特殊结构物。

| 属性 | 说明 |
|------|------|
| **外观** | 形态各异——水晶、光球、石碑、雕像等 |
| **功能** | 驱动整个地下城的运作：生成怪物、维持层间连接、分配资源 |
| **意识** | 每座地下城的核心都具备模糊的自主意识，这些懵懂新生意识虽然有智慧、甚至有些智慧不俗但没见识、没阅历，更没有灵感 |
| **弱点** | 摧毁核心 = 攻克该地下城 = 地下城永久消失 |
| **攻克奖励** | 核心被摧毁时释放大量经验，并产出该城的最高品质奖励 |

### 4.2 核心意识（懵懂新生意识）

核心意识是有本能智慧但缺乏经验阅历的新生意识——它的"智慧"是混沌孕育的结果，不是真正的理性思维，而是**以本能驱动为主的预设模式**：

| 能力 | 说明 |
|------|------|
| **怪物布防** | 按固定模式在层内布置怪物 |
| **难度曲线** | 线性递增，不会因入侵者策略调整 |
| **资源分布** | 随机分布，不受外界影响 |
| **陷阱逻辑** | 简单触发式，无联动设计 |
| **创新能力** | 无——永远不会发明新布局 |

**职业者面对的防守方，永远是"本能型新生意识"级别的对手——战术上可预测、可被利用其固定模式，因为它的"智慧"没有经验支撑。**

> 主角地下城的特殊性（穿越者意识接管核心）详见 [`04_金手指系统.md`](04_金手指系统.md) §三

### 4.3 源质（完整版）

> 本节为源质体系的完整说明。源质的相关机制在金手指系统中仅保留主角特有的"源质转经验"一项，其余通用规则均以本节为准。

#### 4.3.1 定义

源质是所有地下城共有的底层资源——地下城赖以运转和成长的燃料。

#### 4.3.2 来源

源质有三个来源：

| 来源 | 说明 |
|------|------|
| **每日产出** | 地下城核心每天自动产生一定量源质（稳定基础收入） |
| **受伤转化** | 职业者在城内战斗中受伤（无论轻重），逸散的生命能量被地下城吸收。轻度受伤产率约为死亡的1-5%，重度受伤可达10-20% |
| **死亡转化** | 职业者在城内死亡时，产生大量源质（远高于受伤） |

> 源质与经验是两套不相通的体系：经验是职业者的个人成长资源，源质是地下城的运营资源。

#### 4.3.3 用途

| 用途 | 消耗 | 说明 |
|------|------|------|
| **维持运转** | 持续消耗 | 每层、每只怪物、每个陷阱都需要源质维持。规模越大，每日消耗越高 |
| **增加层数** | 大量 | 每增加一层消耗递增。同时核心品质提升 |
| **核心品质提升** | 大量 | 提升地下城核心的品质等级，解锁更多权限 |
| **强化怪物/解锁产出** | 中等 | 提升已有怪物的等级，或定向引导某一类资源产出 |
| **源质→经验转化** | 大量（特殊用途） | 仅主角特有能力，详见图04 |

> 源质由地下城核心使用。职业者无法使用源质。

#### 4.3.4 收支平衡（核心经营挑战）

- **日常收入：** 每日产出 + 玩家受伤产生的流水。
- **意外收入：** 玩家死亡（不作死就不会死）。
- **主要支出：** 维持运转 + 扩张 + 核心品质提升。

地下城经营的核心矛盾：
- 让玩家"有收获但差一点"——既能持续吸引人来刷，又能产生足够的受伤源质。
- 难度太低（玩家无伤通关）→ 无受伤收入；难度太高（玩家频频死亡）→ 吓跑玩家。
- 被攻克 → 全盘皆输。

> 主角地下城的独特经营方式详见 [`04_金手指系统.md`](04_金手指系统.md)

### 4.4 资源产出

职业者在城中击杀怪物、探索场景可获得：
- **经验值**（职业者获得，城无法干预）
- **装备**（从怪物身上掉落或场景宝箱中获取）
- **技能书**（稀有，高阶怪掉落概率增加）
- **材料**（炼金、锻造、附魔所需）
- **特殊凭证**（极稀有，解锁隐藏职业或特殊权限）

资源产出的品质和数量与层数正相关。

> 当前时代，没有人知道产出品的市场价值——大家打到东西后只能凭直觉估价或自己用。

### 4.5 魔物

每阶层的魔物由核心根据预设规律生成。魔物是地下城孕育的生命，并非全按基础本能行动——其智力水平取决于位阶和等级：**低等级、低位阶的魔物才是本能型**（随机游荡、感应范围攻击、无战术配合），**中高端魔物有不同程度的自我意识和战术能力**（初级智能可简单配合，战术级能指挥低阶同类、设伏、判断入侵者职业弱点）。

| 类型 | 特征 |
|------|------|
| **普通魔物** | 数量多，实力中等，主要经验来源 |
| **精英魔物** | 实力远超同阶层普通魔物，掉落更好 |
| **阶层主** | 每阶层的守关者，击败后可进入下一阶层 |
| **核心守卫** | 核心阶层的终极Boss，攻克该城的关键 |

## 五、攻克机制

### 5.1 攻克流程

进入 → 逐层推进 → 击败每层层主 → 抵达核心层 → 摧毁核心 → 攻克成功 → 地下城永久消失

- 没有"反复通关"或"重置刷新"机制。
- 每座城都是一次性资源。
- 队伍可分批进入，但攻克进度以核心层摧毁为准。

### 5.2 队伍规模

| 队伍规模 | 特点 |
|---------|------|
| **单人** | 极少，仅高阶职业者碾压低层城才会尝试 |
| **小队（3-5人）** | 标准配置，各职业搭配，最有效率 |
| **团队（6-15人）** | 大型城的推荐规模 |
| **联合作战** | 多个团队协同攻克超高难度城 |

### 5.3 攻克奖励

攻克的城被摧毁后释放大量经验，参与者的系统会自动结算：
- 等级提升（如果经验足够）
- 技能领悟/升级
- 掉落获取（核心产出的最高品质物品）
- 部分城攻克后会产生"首杀记录"（荣耀性质，无实际增益）

> 当前时代，还没有人类成功攻克过任何一座地下城——职业者还在摸索阶段，连浅层的层主都没打过几回。

### 5.4 攻克失败

| 失败方式 | 后果 |
|---------|------|
| **全员死亡** | 最惨烈，队伍团灭 |
| **被迫撤退** | 队伍主动撤离，下次可继续推进 |
| **连杀被阻** | 某一层始终无法突破，放弃攻略 |

## 六、地下城分类

| 分类维度 | 类型 | 特征 |
|---------|------|------|
| **功能** | **战斗型** | 以战斗为核心，经验产出高 |
| | **资源型** | 材料产出丰富，适合生活职业者 |
| | **混合型** | 最常见的类型，均衡产出 |
| | **特殊型** | 存在特殊机制或隐藏条件（极稀有） |
| **强度** | **小型** | 6-20层，适合未觉醒的非职业者、一转1-10级，催化普通人觉醒 |
| | **中型** | 21-50层，适合二转、三转 |
| | **大型** | 51-90层，适合三转、四转 |
| | **超大型** | 91层以上，四转及四转以上可挑战 |
| **状态** | **萌发** | 仍在混沌中孕育，层数持续增长 |
| | **已降临** | 固定在蓝星某处，层数不变 |
| | **被攻克** | 核心已毁，永久消失 |

## 七、地下城与社会（初期混乱状态）

> 当前社会背景：超凡出现仅约3周，社会对地下城的认知几乎为零。以下描述的是初期混沌状态，不是成熟期的社会结构。

### 7.1 政府管控（初期）

**没有统一的管控体系。**

- **没有觉醒者管理局**——不存在任何官方机构专门管理觉醒者和地下城。
- **没有统一的出入登记制度**——找到入口就可以进，无人登记无人阻拦。
- **各地各自为政：**
  - 有觉醒者出现的地方，当地政府紧急组织临时应对。
  - 没有觉醒者的地方，地下城就放着不管。
  - 没有中央协调，省与省之间、城与城之间各自为战。

**实际状态（不是系统性封锁）：**
- 城市内降临 → 当地紧急拉警戒线/围起来，但不知道下一步该做什么——不是"管控"，更像是"不知如何处理所以先围上"。
- 荒野降临 → 可能被目击者上报，也可能长时间无人知晓。
- 偏远地区降临 → 数周甚至数月后才被发现。
- 不同地区的应对差异极大——有的封路、有的一圈人围着看了几天、有的干脆当没看见。

### 7.2 地下城经济（萌芽期）

**没有成型的经济体系。**

- **没有定价体系：** 产出品没有市场价。大家凭感觉交易——"这把剑我觉得值这个价，你觉得呢？"
- **几乎不存在公开交易：** 封锁/信息混沌环境下不存在成规模的交易活动。
  - 极少数私下交换在隐蔽场所进行（网吧角落、小吃店碰头），而非公开摆摊。
  - 交易形式以物易物，无定价标准。
- **信息不流通：**
  - 没有情报交易市场。
  - 某座城内部什么样，只有进去过的人才知道，但没有分享渠道。
  - 偶尔有人在论坛发帖说某城有危险，但真假无人验证。
- **无资本介入：** 大资本还在观望。没有人知道"地下城产出"到底值多少钱、有没有长期价值。

### 7.3 信息层级（不存在的分层）

**当前信息不存在层级分化的条件。**

| 维度 | 现状 |
|------|------|
| **城的位置** | 仅通过口耳相传。目击者上报/论坛发帖/朋友圈转发。没有官方清单 |
| **层数** | 没人知道具体层数，除非有人进去探过 |
| **内部情报** | 进去探过的人凭记忆分享，很可能不准确 |
| **产出价值** | 无法评估。打出来的东西没有市场参考价 |
| **攻略** | 不存在。没有人积累了足够经验来写攻略 |
