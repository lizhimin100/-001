# 01 世界观（世界的基础面貌与常识——所有创作中需要结合的世界观上下文。）

时间：超凡出现后约3周。
全世界还在混乱中摸索。

---

## 一、世界概况

### 1.1 世界名称

蓝星。

### 1.2 基本格局

202X年，超凡降临。异空间，地下城，活体秘境，总之从混沌中孕育生成的超凡之物，陆续降临蓝星。

世界的运行底层是**半数据化**。职业者看到的是面板、经验值、技能树，数据背后是真真实实的超凡能量。边界模糊，没人搞懂"这到底是个游戏还是现实"。

当前时间：超凡出现后约3周。

各国政府还在混乱中，没有统一的地下城应对策略。觉醒者是散乱的、自发的。没有人知道发生了什么，那些突然出现的"异空间"是什么、有什么功能。职业体系不成熟，更没人能判断职业品级和稀有度。

社会结构没崩——大多数人还在过自己的日子——但根本性的改变已经开始了。

### 1.3 时代特征

超凡出现初期（约3周内）：

- **信息混沌。** 没有公开承认，没有全面封锁。普通人处于"半信半疑"——有人刷到过视频，官方不回应、热搜被删，没人能确认发生了什么，也没人能否认。
- **正文约束：** 正文前期不使用"地下城"一词。只说"地基异常""封路了"，角色之间用"异空间""那个地方""秘境"等模糊称谓。没有人知道这些东西真正的名字。
- 全民隐约知道"有事发生了"，但形不成统一认知。很多人还在观望或选择不信。
- 觉醒者极少，职业者属于极端稀缺战力。
- 仍有地下城在异空间混沌中孕育，部分刚刚降临固定。
- **最小的地下城也有6层。** 几乎没有低于6层的城。
- **没有统一的觉醒者管理机构。** 觉醒者分散各地，靠自发性组织或个人行动。
- **没有地下城出入登记制度。** 找到入口就能进。
- 各地地下城信息不流通，没有情报交易市场。你只能听说"某地出现了一个地下城"，但不知道里面什么样。
- 产出品没有定价体系。全凭感觉交易。

## 二、地下城

> 完整设定见：[`03_地下城设定.md`](03_地下城设定.md)
> 本节是快速参照。

### 2.1 定义

地下城是超凡出现的主产物。每座城由若干层构成——每层一个独立空间，越深层的怪物越强、奖励越丰厚。

### 2.2 核心常识

| 条目 | 说明 |
|------|------|
| **生成方式** | 异空间混沌中孕育，核心意识成熟后自动现世，固定降临蓝星某位置 |
| **/ | **降临不稳定期** | 城在降临初期（尤其是首降后冷却期前）核心意识不稳定，门可能不受控地自行开启。此时门轴无声转开，主动吸入附近的生物（"区域置换"而非物理吸力）。此状态随第一次降临结束而消退 |
| **攻克即消失** | 摧毁核心=攻克=地下城永久消失，一次性机制 |
| **核心意识** | 每座城有本能AI，自动管理布防与产出 |
| **/ | **入口机制** | 入口无法关闭——在不稳定期入口还有可能会不受控自开，吸引生命进入地下城（区域置换）；注意：入口永久开放，固定存在 |
| **源质** | 地下城的成长燃料，死去的生命被吸收转化而成 |
| **经验** | 职业者个人资源，杀怪探索获得，与源质两套体系 |
| **降临** | 定型后固定在蓝星某位置，不可移动。当前处于陆续降临阶段 |

## 三、职业者

### 3.1 觉醒

接触地下城或受超凡能量浸染后，部分人类觉醒职业。觉醒过程不可控，人群分布无规律。

**地下城是觉醒的核心入口。** 进入地下城一定时间，或接触其内部的超凡之力，可刺激普通人的职业资质，使其在场内当场觉醒。

> 当前人们对这些突然出现的空间没有统一称谓——有人叫"异空间"、"秘境"、"那个地方"，甚至"地狱之门"。还没有人知道它们真正的名字。

觉醒后获得：
- 职业名称
- 技能树
- 属性面板（私密，可主动展示）

#### 觉醒的稀有度

- 觉醒者占总人口比例极低。
- 当前无官方机构也无监管体系，觉醒者处于自发状态。
- 职业的成长必须在地下城中进行。

### 3.2 当前情况

- 觉醒者占总人口比例极低——极端稀缺资源。
- **没有统一的觉醒者管理机构。** 觉醒者自组织，各地情况差异巨大。
- 有些地方临时组织了觉醒者小队，有些觉醒者自己组野队，大多数还在独自摸索。
- 职业体系不成熟。大家只知道几个大分类（战士/法师/辅助/生活/特殊），具体职业变体、技能树、强度评估——没人能说清楚。
- 觉醒者成长必须在地下城中进行，但很多地下城还没被充分探索。

### 3.3 职业类型

| 类型 | 特征 |
|------|------|
| 战斗系 | 战士、法师、刺客、弓箭手、骑士等 |
| 辅助系 | 治疗、防御、增益等 |
| 生活系 | 工匠、药师、鉴定师、采集者等 |
| 特殊系 | 极稀有，千奇百怪。领主系是其重要分支——以领地或据点为核心的经营型职业 |

> 当前时代，特殊系几乎不被大众所知。绝大多数人见过的只有战斗系和辅助系。

## 四、社会结构

### 4.1 初期混乱状态

**没有统一的觉醒者管理体系。** 各地应对方式差异巨大：

- **大城市：** 地方政府紧急组织临时应对小组，少数觉醒者和警察/军队配合，尝试探索辖区内的地下城。没有成熟经验，边试边错。
- **小城市/乡镇：** 有觉醒者出现的地方自发管理；没有觉醒者的地方，地下城放着不管。
- **总体：** 政府在努力维持秩序，但面对"地下城"这种前所未见的现象，没有任何国家有成熟方案。

**地下城出入状况：**
- 没有统一的出入登记制度。
- 找到入口就能进。
- 有些地下城出现在偏僻地方，好几天都没人发现。

### 4.2 民间势力（萌芽期）

- **自由职业者：** 第一批觉醒者中的大多数。独自探索或临时组队。
- **临时交易：** 地下城附近出现的自发交易。凭感觉估价——"这把剑我打的，你觉得值多少就换多少。"
- **信息传播：** 口耳相传。可能有人在论坛发帖说某座地下城有危险，没人验证真假。
- **资本势力：** 还在观望。大资本没入场，所有人都还不确定这个"地下城产业"到底有多大价值。
