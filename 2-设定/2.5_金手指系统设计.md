# 2.5 职业设定：地下城主

> 其他所有地下城的防守方都是AI，只有主角的后台坐着一个真正的人。

---

## 一、职业本质

### 1.1 定义

**地下城主**是特殊系·领主分支下的一个变体职业。

领主类职业者的核心能力与"领地"或"据点"的经营管理相关——有的领主掌控一座要塞，有的领主管理一片区域，有的领主的领地表现为追随者。而主角的领地，是一座地下城。

其他职业者的目标都是进入地下城、推进、攻克、获取资源。地下城主不同：他是这座地下城的**管理者**。

### 1.2 觉醒根源

李冠阳在首次进入地下城前，对自己的觉醒方向没有预判。职业由灵魂特质在灵气刺激下自然显现——战士觉醒是因为灵魂骨子里倾向战斗，法师觉醒是因为对元素亲和，辅助觉醒是因为治愈本能的释放。

他的灵魂深处有一个异常——转世印记。但这不是决定性因素。

真正关键的是两件事同时发生：

① 他的觉醒方向是**特殊系·领主分支**。领主类职业本来就极其罕见，而他恰好拥有这个资质。

② 他第一次进入的是一座**未定型地下城**——这城在混沌中孕育已久，但还未定型降临。未定型意味着它没有固定的核心意识，没有固化的层数结构——它还是"空白的"，可以被人接手。

一个拥有领主资质的觉醒者，在一座尚未封闭的空白地下城里觉醒。职业类型与地下城环境的相互作用，使他的领主类能力找到了一个前所未有的具体化方向——不是要塞、不是领地、不是追随者，而是**这座地下城本身**。

不是灵魂自己凿路。是种子落对了土。

### 1.3 唯一性

领主类职业本身就是特殊系中极罕见的分支。而"地下城主"作为领主类的具体变体，还需要两个前提条件同时满足——觉醒者拥有领主资质，且觉醒当时正身在一座未定型的地下城中。

蓝星上不存在第二个同时满足这两个条件的人。

**主角是目前唯一的地下城主。**

### 1.4 灵感来源

这个设定的灵感来自日本漫画《怪物大师》（Beastars 作者板垣巴留的早期作品）中关于"与怪物签订契约的少年"的概念——人与非人之物之间的绑定关系，以及由此产生的身份困境。主角与地下城的绑定在形式上与此有共通之处：一个个体被永续地、不可逆转地与一个非人存在连接在一起，无法割舍，无法转让。

然而本质不同在于，主角的绑定不是契约精神或等价交换的结果，而是**觉醒方向与自身环境的一次完美耦合**——该来的时候，来了该来的人，进了该进的地方。

## 二、核心机制：绑定

### 2.1 绑定过程

职业觉醒的瞬间，未定型地下城的核心——混沌的、尚未形成完整意识的底层结构——与主角的领主资质产生共振，建立深层连接。

绑定不可解除，不可更换。

### 2.2 绑定结果

| 结果 | 说明 |
|------|------|
| **永久归属** | 该地下城的管理权限归属主角，其他人在系统层面无法夺走 |
| **同步感知** | 主角可随时感知地下城的状态——层数、怪物、资源、是否有人进入 |
| **绑定风险** | 地下城被攻克 → 核心被摧毁 → **主角死亡**（详§6.2） |

### 2.3 关联权限

绑定后，主角获得该地下城的后台管理面板。面板内容：

| 模块 | 功能 |
|------|------|
| **总览** | 当前层数、源质储量、怪物总数、产出清单 |
| **怪物管理** | 查看/调整各层怪物种类、数量、分布、行为逻辑 |
| **地形编辑** | 调整通道布局、陷阱位置、房间功能 |
| **资源产出** | 设定各层产出方向（装备/技能书/材料等） |
| **入城记录** | 查看进入者的职业、等级、战损数据 |
| **降临控制** | 发起降临、收回降临（详见第四章） |

## 三、后台操控

### 3.1 运作方式

主角不在"实地"跑图面怪。他的工作是在后台进行战略部署——

```
玩家视角：进入地下城 → 打怪 → 开路 → 决策 → 推进
主角视角：看面板 → 分析数据 → 调布局 → 看结果 → 优化
```

### 3.2 可操控项目

| 操控项 | 初期 | 中期 | 后期 |
|--------|------|------|------|
| **入口位置** | 微调（±50米） | 可移动至可降临范围内任意位置 | 同上 |
| **第一层怪物** | 选种类、数量 | 多层可调 | 全部可调 |
| **陷阱** | 基础陷阱（预设模板） | 复合陷阱（多层联动） | 自定义陷阱逻辑 |
| **怪物行为** | 巡逻/原地待命 | 战术行为（埋伏/追击/撤退） | 高级AI逻辑链 |
| **资源产出** | 随机 | 定向引导 | 精准控制掉率 |
| **区域规则** | 无 | 基础buff/debuff | 自定义规则 |

### 3.3 人脑 vs AI 的差异化

| 对比项 | 普通地下城（AI） | 主角地下城（人脑） |
|--------|----------------|------------------|
| **怪物布防** | 固定模式、可预测 | 因时制宜、实时调整 |
| **难度曲线** | 线性递增 | 可根据玩家数据优化 |
| **陷阱逻辑** | 简单触发式 | 可设计连环/诱骗/反套路 |
| **资源产出** | 随机分布 | 可引导定向产出 |
| **适应性** | 无——不会针对玩家策略调整 | 有——会针对热门职业/套路调整克制布局 |
| **创新能力** | 无——永远不会发明新布局 | 有——可以设计出AI永远想不到的配置 |

## 四、特殊性：层数不固定 + 可移动

主角绑定的地下城有两项所有其他地下城不具备的特性。

### 4.1 层数不固定

普通地下城在混沌中孕育至定型后降临蓝星。层数固定，终身不变。

主角的地下城**永远不会定型**。因为绑定发生时它尚未定型，而主角的经营权限接管了后续的层数增长。只要源质充足，层数可以持续增长。

| 阶段 | 层数 | 说明 |
|------|------|------|
| 绑定初始 | 1~3层 | 刚绑定时的状态 |
| 初步发育 | 3~6层 | 积累一定源质后的规模 |
| 中期 | 6~12层 | 稳定运营后的规模 |
| 后期 | 12层+ | 大量源质投入后的规模 |

#### 开新层机制：开盲盒

新增一层的流程如下：

```
积攒源质 → 确认底层边界是否有空间 → 支付扩容费用
→ 混沌随机生成层内地形 → 玩家看到新层开放 → 主角用面板微调
```

开新层时，层内的基础结构和怪物类型由混沌中的残余波动随机决定，具有高度不确定性：

| 盲盒维度 | 方差 | 实际影响 |
|---------|:---:|---------|
| **地形** | 大 | 迷宫、平原、洞穴、古城废墟……纯随机 |
| **怪物种类** | 大 | 野兽、亡灵、元素生物、异界造物——无法预知 |
| **附加属性** | 中 | 该层是否附带特殊debuff/环境伤害/隐藏资源 |
| **地形大小** | 中 | 从足球场到一整个街区大小的空间不等 |
| **隐藏要素** | 小 | 是否有隐藏Boss、隐秘通道、特殊事件触发点 |

主角无法控制混沌的生成结果，只能在新层生成后通过面板做有限微调（移走部分怪物、调整难度曲线、定向产出）。**运气也是经营的一部分。**

### 4.2 可移动（降临/收回）

普通地下城降临蓝星后固定在原地，不可移动，不可隐藏。

主角的地下城可以：

| 操作 | 效果 | 冷却/限制 |
|------|------|----------|
| **降临** | 地下城在蓝星上出现，入口对职业者开放 | 需要消耗源质 |
| **收回** | 地下城从蓝星消失，回到异空间待命 | 无冷却，但收回期间不产生收益 |
| **移动** | 在另一个位置重新降临 | 新位置需在可降临范围内 |

**战略意义**：
- 可躲避强敌——发现被盯上就收回跑路
- 可更换市场——一个位置玩家刷腻了，换个新地方重新开张
- 可伏击——引诱特定目标进入后收回入口，形成瓮中捉鳖

### 4.3 限制条件

| 限制项 | 说明 |
|--------|------|
| **降临范围** | 不能无限远，以主角当前位置为中心一定半径内 |
| **源质消耗** | 降临和移动均消耗源质，层数越高消耗越大 |
| **收回时间** | 收回后下次降临需等待一定时间 |
| **唯一性** | 一次只能在一个位置降临，不能分身 |

## 五、源质体系

### 5.1 定义

源质是地下城的底层资源。降临期间，在地下城内死去的生命（无论职业者还是怪物）被地下城吸收，转化为源质。

**受伤产源质**：不限于死亡——职业者在城内战斗中受伤（无论轻重），流淌出的血液和逸散的生命能量同样会被吸收，只是产率远低于死亡。单人轻度受伤的源质产出约为死亡的1-5%，重度受伤可达25-50%。这种机制确保了即便玩家没有死在城中，每一次战斗消耗仍会为城带来微量收益。

### 5.2 用途

| 用途 | 消耗 | 说明 |
|------|------|------|
| **增加层数** | 大量 | 每增加一层消耗递增 |
| **强化怪物** | 中等 | 提升已有怪物的等级/能力 |
| **解锁产出** | 中等 | 定向引导某一类产出 |
| **降临/移动** | 按次消耗 | 层数越高消耗越大 |
| **解锁权限** | 累积门槛 | 达到一定总量后解锁新操控权限 |

### 5.3 收支平衡（核心经营挑战）

主角需要管理：
- **收入**：入城职业者与怪物厮杀产生的战损源质
- **支出**：扩容、强化、移动的成本
- **最优策略**：让玩家"有收获但差一点"——既能持续吸引人来刷，又能制造足够的战损来维持源质收入

杀多了→玩家被劝退→没有收入来源
杀少了→源质增长慢→发育太慢
被攻克了→全盘皆输

## 六、风险与失败条件

### 6.1 核心风险

| 风险 | 后果 | 发生条件 |
|------|------|---------|
| **地下城被攻克** | **主角死亡**——城毁人亡 | 有队伍杀穿所有层并摧毁核心 |
| **身份暴露** | 被政府/势力/个人盯上 | 后台操作过于明显引起怀疑 |
| **经营失衡** | 源质枯竭，无法继续发育 | 长期入不敷出 |
| **降临被蹲** | 收回前被大量高阶职业者堵门强攻 | 位置已暴露且未能及时收回 |

### 6.2 被攻克 = 死亡

地下城被攻克→核心被摧毁→绑定彻底断裂→**主角死亡**。

城核绑定机制（详见 `2.4_力量体系.md §5.4.2`）决定了：主角与地下城已是生命层面的连接。城破人亡，不可挽回。

- 不存在"活着失去职业"的可能性——攻克=死亡
- 绑定消失是死亡的结果，不是前提
- 不存在任何已知手段可以解除绑定而不死（即使神明也无法干涉城核绑定）

**这一设定是主角经营策略的底层红线**：他的每一次布局，每一场引诱，每一次冒险，赌的都是全副身家性命。

## 七、职业定位总结

```
不是战斗系——主角不是靠拳头说话的
不是辅助系——主角不给人加buff
不是生活系——主角不会生产装备和药剂
是特殊系·领主分支——以领地为核心的经营管理型职业

地下城主 = 领主类 · 地下城环境变体

他的战场不在怪物堆里，而在后台面板前。
他的武器不是刀剑法术，而是数据分析和战略布局。
他的成长不靠打怪升级，而靠地下城扩容和权限解锁。
```
