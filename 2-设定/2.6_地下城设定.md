# 2.6 地下城设定

> 地下城——灵气复苏的主产物，职业者的试炼场与资源来源。
> 本文档为地下城作为世界现象的全貌设定，独立于任何单一个体。

---

## 一、地下城是什么

### 1.1 定义

地下城是**活体秘境**——在异空间混沌中生成的独立空间结构。每座地下城由若干层构成，每层一个独立空间，越深层的怪物越强、奖励越丰厚。

地下城不是"副本"——不是系统生成的临时关卡，而是**真实存在的异空间结构**，被攻克后永久消失，不会重置刷新。

### 1.2 地下城与世界的关系

```
异空间混沌（孕育期）
      ↓ 混沌停止、层数固定
降临蓝星（固定在某一地理位置）
      ↓ 职业者进入挑战
被攻克或持续存在
```

- 地下城是灵气复苏的**产物**，不是原因
- 地下城生成于异空间的混沌中，混沌中孕育的"种子"决定了城的类型和基础属性
- 降临后固定在蓝星某处，不可移动（注：主角的绑定城是唯一特例）

### 1.3 地下城与觉醒

D级地下城在江湖中俗称"**觉醒坑**"——它是普通人晋升职业者的关键入口。

- D级城释放的微量灵气能刺激普通人潜藏的职业资质，使部分人在城内战斗时当场觉醒
- 觉醒后立刻获得1级学徒阶，但由于已经身在城中，需要杀穿当前层才能出去
- 因此D级城的实际死亡率不低——觉醒前的普通人太过脆弱，战斗中的任何失误都可能导致死亡
- 国家允许D级城存在，但不鼓励未成年人和无资质者擅自进入
- 本质上，D级城是整个灵气复苏体系的"入口"——没有它，觉醒就只能是少数先天资质者的特权

## 二、生成与降临

### 2.1 混沌孕育期

每座地下城在异空间的混沌中孕育。此阶段：
- 层数持续增长——越早开始孕育的城，成型时层数越多
- 混沌中孕育的类型决定城的分类（战斗型/资源型/迷宫型等）
- 孕育期长短不一，从数月到数十年都有
- 孕育期间外界无法探查，只有少数觉醒者能隐约感知到异空间的波动

### 2.2 定型降临

当混沌停止、层数固定时，地下城"降临"蓝星现实世界：
- 降临位置不可预测——可能出现在荒野、城市地下、海洋、天空
- 降临时会引发灵气波动，强力的城甚至有天象异变
- 降临后层数**不再变化**，位置**不可移动**
- 各国的觉醒者管理机构会监控降临事件并迅速封锁现场

### 2.3 降临类型

| 类型 | 说明 | 应对方式 |
|------|------|---------|
| **自然降临** | 孕育完成后自发降临 | 最常见，波动可被监测 |
| **诱导降临** | 通过特殊手段提前触发 | 极稀有，多为大势力操作 |
| **连锁降临** | 某城被攻克后，附近混沌中相邻的城被触发降临 | 间歇期会形成"地下城群" |

### 2.4 降临优先级

政府会根据地下城降临的位置和预估强度采取不同策略：
- **野外降临**：开放探索，职业者自由组队进入
- **城市降临**：紧急封锁，评估后决定是攻克还是绕行
- **高危降临**：预估层数过多或怪物过强，直接划为禁区，高阶职业者联合处理

## 三、层数结构

### 3.1 基本规则

地下城由多层构成。每层一个独立空间，空间大小不一。

| 属性 | 说明 |
|------|------|
| **层数范围** | 3-50层不等，视孕育时长而定 |
| **层间连接** | 层与层之间通过传送阵或通道相连 |
| **层内空间** | 少则足球场大小，多则整座城市规模 |
| **越层机制** | 部分特殊道具允许跳过中间层（极稀有） |

### 3.2 层数分布规律

| 区域 | 层数范围 | 难度特征 |
|------|---------|---------|
| **浅层** | 前30%的层 | 适合学徒阶，怪物强度低，基础产出 |
| **中层** | 中间40%的层 | 适合超凡阶，难度爬坡，产出品质提升 |
| **深层** | 后25%的层 | 大师阶起步，高阶怪物，稀有资源 |
| **核心层** | 最后1层 | 地下城核心所在，最强Boss守卫，攻克关键 |

### 3.3 层数命名规范

各层通常按顺序编号（第1层、第2层……），职业者之间约定俗成：
- 1-5层称为"表层"
- 6-15层称为"中层"
- 16层+称为"深层"
- 最后一层称为"核心层"

## 四、地下城的构成要素

### 4.1 核心

每座地下城都有一个**核心**——位于最深处的特殊结构物。

| 属性 | 说明 |
|------|------|
| **外观** | 形态各异——水晶、光球、石碑、雕像等 |
| **功能** | 驱动整个地下城的运作：生成怪物、维持层间连接、分配资源 |
| **意识** | 每座地下城的核心都具备模糊的自主意识——类似本能驱动的人工智能 |
| **弱点** | 摧毁核心 = 攻克该地下城 = 地下城永久消失 |
| **攻克奖励** | 核心被摧毁时释放大量灵气和经验，并产出该城的最高品质奖励 |

### 4.2 核心意识

核心意识不是真正的智慧，而是**预设模式的组合**：

| 能力 | 说明 |
|------|------|
| **怪物布防** | 按固定模式在层内布置怪物 |
| **难度曲线** | 线性递增，不会因玩家策略调整 |
| **资源分布** | 随机分布，不受外界影响 |
| **陷阱逻辑** | 简单触发式，无联动设计 |
| **创新能力** | 无——永远不会发明新布局 |

**职业者面对的防守方，永远是AI级别的对手。**
（注：主角绑定的城中，这些职能由人脑接管——详见 `2.5_金手指系统设计.md`）

### 4.3 源质

| 属性 | 说明 |
|------|------|
| **定义** | 所有地下城共有的底层资源 |
| **产生** | 降临期间，在地下城内死去的生命（无论职业者还是怪物）被地下城吸收转化为源质 |
| **用途** | 驱动地下城扩容：增加层数、提升怪物强度、提升产出品质 |
| **使用者** | 仅地下城核心可以使用源质。职业者无法使用源质 |
| **与经验的区别** | 经验是职业者的个人成长资源，源质是地下城的运营资源——两套不相通的体系 |

### 4.4 资源产出

职业者在城中击杀怪物、探索场景可获得：
- **经验值**（职业者获得，城无法干预）
- **装备**（从怪物身上掉落或场景宝箱中获取）
- **技能书**（稀有，高阶怪掉落概率增加）
- **材料**（炼金、锻造、附魔所需）
- **特殊凭证**（极稀有，解锁隐藏职业或特殊权限）

资源产出的品质和数量与层数正相关。

### 4.5 怪物

每层的怪物由核心根据预设规律生成：

| 类型 | 特征 |
|------|------|
| **普通怪** | 数量多，实力中等，主要经验来源 |
| **精英怪** | 实力远超同层普通怪，掉落更好 |
| **层主** | 每层的守关者，击败后可进入下一层 |
| **核心守卫** | 核心层的终极Boss，攻克该城的关键 |

## 五、攻克机制

### 5.1 攻克流程

```
进入 → 逐层推进 → 击败每层层主 → 抵达核心层 
→ 摧毁核心 → 攻克成功 → 地下城永久消失
```

- 没有"反复通关"或"重置刷新"机制
- 每座城都是一次性资源
- 队伍可分批进入，但攻克进度以核心层摧毁为准

### 5.2 队伍规模

| 队伍规模 | 特点 |
|---------|------|
| **单人** | 极少，仅高阶职业者碾压低层城才会尝试 |
| **小队（3-5人）** | 标准配置，各职业搭配，最有效率 |
| **团队（6-15人）** | 大型城的推荐规模 |
| **联合作战** | 多个团队协同攻克超高难度城 |

### 5.3 攻克奖励

攻克的城被摧毁后释放大量灵气和经验，参与者的系统会自动结算：
- 等级提升（如果经验足够）
- 技能领悟/升级
- 掉落获取（核心产出的最高品质物品）
- 部分城攻克后会产生"首杀记录"（荣耀性质，无实际增益）

### 5.4 攻克失败

| 失败方式 | 后果 |
|---------|------|
| **全员死亡** | 最惨烈，队伍团灭 |
| **被迫撤退** | 队伍主动撤离，下次可继续推进 |
| **连杀被阻** | 某一层始终无法突破，放弃攻略 |

## 六、地下城分类

| 分类维度 | 类型 | 特征 |
|---------|------|------|
| **功能** | **战斗型** | 以战斗为核心，经验产出高 |
| | **资源型** | 材料产出丰富，适合生活职业者 |
| | **混合型** | 最常见的类型，均衡产出 |
| | **特殊型** | 存在特殊机制或隐藏条件（极稀有） |
| **强度** | **D级** | 不入阶觉醒池，1-3层，催化普通人觉醒 |
| | **C级** | 学徒阶可挑战，4-6层 |
| | **B级** | 超凡阶门槛，7-12层 |
| | **A级** | 大师阶起步，13-18层 |
| | **S级** | 传奇级，19层以上 |
| **状态** | **未定型** | 仍在混沌中孕育，层数持续增长 |
| | **已降临** | 固定在蓝星某处，层数不变 |
| | **被攻克** | 核心已毁，永久消失 |

## 七、地下城与社会

### 7.1 政府管控

- 觉醒者管理机构负责监控地下城降临
- 各地下城出入受管控，未经登记的职业者不得擅自进入
- 产出部分收归国有，部分允许民间交易
- 攻克优先级由政府按城市安全和资源需求统一调度

### 7.2 地下城经济

围绕地下城已形成新兴产业链：
- **攻略队**：职业者组队，专门攻克或刷特定城
- **情报交易**：各城内部布局、Boss情报、产出数据
- **装备/材料市场**：产出品交易，带动炼金、锻造等生活系
- **城际物流**：不同城产出的跨区调配

### 7.3 信息层级

| 层级 | 公开信息 | 使用对象 |
|------|---------|---------|
| **公开** | 城的位置、层数、基础难度评估 | 所有职业者 |
| **攻略** | 怪物分布、Boss信息、产出清单 | 付费情报 |
| **深层** | 核心层细节、隐藏机制、首杀记录 | 高阶职业者/大势力内部 |

## 八、地下城与力量体系的关系

详见 `2.4_力量体系.md`。核心规则：**同阶进同城，越级危险翻倍**。

| 职业者阶位 | 可挑战对应等级 |
|-----------|--------------|
| 不入阶 | D级 |
| 学徒阶 | C级 |
| 超凡阶 | B级 |
| 大师阶 | A级 |
| 传奇阶 | S级 |

越级进入更高级地下城的具体存活率，详见 `2.4_力量体系.md §1.1` 四级风险量表。

地下城是职业者的**主要升级途径**。不在城中战斗，升级速度极慢。

## 十、写作运用指南

- **地下城不是副本**：强调一次性、不可重复、真实消逝的性质
- **信息差是资源**：知道某城内情 vs 不知道，战斗效率和伤亡差距巨大
- **攻克是事件**：一座重要城被攻克时，会引发经济和社会层面的连锁反应
- **未定型城是战略资产**：谁掌握了未定型城的情报，谁就有先发优势
