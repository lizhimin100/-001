# 04 金手指系统（主角专属——双面板结构，职业者面板+地下城核心操控面板。）

本文档只写主角的专属机制。
通用设定（地下城AI规则、源质通用规则、社会状态等）见各对应文件。

---

## 一、双面板结构

### 1.1 定义

李冠阳拥有**两个面板**，分属两个不同的系统身份：

| 面板 | 对应身份 | 功能 |
|------|---------|------|
| **职业者面板** | 明面身份——觉醒职业"法师" | 正常的职业者系统：等级、技能、经验值、属性 |
| **核心操控面板** | 暗面身份——地下城核心分身的操控界面 | 管理地下城：源质收支、层数规划、怪物布置、产出配置 |

> 详细角色信息见 [`../2-角色卡/李冠阳角色卡.md`](../2-角色卡/李冠阳角色卡.md)

### 1.2 觉醒根源

双面板的根源：

李冠阳转世时，灵魂与世界底层规则发生了异常交互——灵魂被分裂成两个独立个体：

- **主体（本体）**：以"转世者"身份在现实世界中生存，觉醒获得正常的职业者面板
- **分身（核心意识）**：被嵌入异空间混沌中，成为一座正在孕育的地下城的核心意识，拥有核心操控面板

两者本质上是**同一个人灵魂的两面**，共享同一意识，但位于两个不同的位置。

### 1.3 唯一性

双面板结构是**独一无二的**——世界上不存在第二个拥有双面板的人类，也不存在第二个拥有分身意识的地下城。

其他地下城的核心意识是预设AI模式组合，不具备真正的智慧——只有主角的分身例外。

> 普通地下城核心的意识运作方式详见 [`03_地下城设定.md`](03_地下城设定.md) §4.2

### 1.4 分身本质

- 分身是**灵魂的另一半**，不是独立人格——两者共享同一个意识（李冠阳的思维）
- 分身没有自己的"人格"或"意愿"——就是李冠阳自己，只是位于地下城核心中
- 分身无法独立行动、无法接管身体、无法脱离核心——它就是那座地下城
- 日常状态：分身大部分时间在"后台"自动运作（执行预设策略），需要时李冠阳可以将注意力切换到分身视角进行操作

## 二、面板机制

### 2.1 独立性

两个面板在系统层面是**独立注册**的：

- 职业者面板：和其他所有职业者一样
- 核心操控面板：由分身的核心提供，是地下城核心的数据面板

这意味着：
- 职业者面板上的等级、技能、经验——都是真实的（不是伪装的）
- 核心操控面板上的源质、层数、怪物——也是真实的地下城经营数据
- 两个面板不会互相冲突，也不会被任何检测手段交叉追踪

### 2.2 互通规则

| 互通类型 | 说明 |
|---------|------|
| **意识同步** | 主体和分身共享同一个意识——主体知道分身看到的一切，分身知道主体想什么 |
| **经验不互通** | 主体通过职业者面板获得的经验属于职业者，不会自动流入分身。反之亦然 |
| **信息互通** | 主体可以通过面板查看地下城的分身状态，分身也可以感知到主体周围的环境 |
| **时间感知** | 两者时间流速一致。不存在"一方时间变慢/变快"的情况 |

### 2.3 日常状态

- 分身地下城在没有外部操作时，按默认预设模式自动运作（维持现有层数、生成怪物、分配资源等）
- 主体白天过正常生活（上课/打工/找地下城探索），空闲时在脑中查看地下城状态
- 切换注意力到分身视角需要集中精神，不能在被监控或战斗中进行

## 三、操控权限

### 3.1 职业技能（本职职业者面板）

主角明面职业是**法师**，觉醒后获得法师职业的基础技能树。

法师是战斗系中的常见远程输出职业：
- 他可以用法师身份参与地下城探索，和其他觉醒者组队
- 他的输出能力合理，不会因为"有金手指就成全能天才"
- 他必须亲自组队打怪升级，不能仅靠地下城分身成长

> 法师的具体技能树写法见角色卡与[`02_力量体系.md`](02_力量体系.md)

### 3.2 操控项（核心操控面板）

主角通过分身面板对地下城进行以下操控（按等级逐步解锁）：

| 操控项 | 解锁条件 | 功能说明 |
|--------|---------|---------|
| **基础监控** | 初始可用 | 查看地下城当前状态：层数、怪物存活、源质存量、玩家位置 |
| **怪物布置** | 初始可用 | 在已开放的层内调整怪物的类型、数量和分布 |
| **难度调整** | 初始可用 | 在已开放的层内微调怪物的强度（上下浮动约30%） |
| **源质分配** | 核心品质达到[待定] | 决定源质的优先级：维持运转/扩张层数/强化怪物/解锁产出 |
| **增加层数** | 核心品质达到[待定] | 消耗大量源质增加新层，同时提升核心品质 |
| **核心品质提升** | 源质充足 | 提升核心品质等级，解锁更多操控权限和功能 |
| **定向产出** | 核心品质达到[待定] | 引导某一类资源定向产出（如增加某类装备的掉率） |
| **陷阱设计** | 核心品质达到[待定] | 在层内自定义陷阱的位置和触发条件 |
| **玩家标记** | 核心品质达到[待定] | 识别进入地下城的玩家职业和等级（但不能查看详细面板） |

> 具体解锁等级和核心品质对应表待定，写作过程中根据进度填充。

操控界面以**视觉化"地图+状态面板"**的形式呈现在主角脑海中——他看到的不是代码和数字，而是宛如游戏界面的脑内画面。

### 3.3 人脑 vs 核心：操控差异

| 维度 | 普通地下城（AI核心） | 主角分身（人脑核心） |
|------|-------------------|------------------|
| **布防模式** | 固定预设模式，不会变化 | 可根据进入者的实力动态调整 |
| **难度曲线** | 线性递增 | 可人为微调 |
| **陷阱逻辑** | 简单触发式 | 可设计联动陷阱 |
| **创新能力** | 无 | 有人类的创造力 |
| **局限性** | — | 需要精力投入，无法24小时专注操控 |

### 3.4 "阴险防守"的来源

主角地下城的防守方式与普通地下城的本质差异在于：**他是人，不是AI。**

普通地下城的怪物布防按固定模式排列——只要职业者找到规律，就可以用最小代价打通每一层。

主角的地下城不同：
- 他可以根据进入者的职业配置动态调整难度（坑队形不合理的小白）
- 他可以设计伏击和陷阱联动（提前想好"如果敌人从A方向来，怎么让B方向的精英怪包抄"）
- 他可以控制怪物的"演技"（让第一波看起来弱，引诱深入后再露出真实难度）

这些在前期不会全部出现——主角需要逐步学习如何像一个"游戏设计师"那样经营自己的地下城。

## 四、特殊性：层数不固定

### 4.1 初始状态：1层雏形

当主角的分身被嵌入混沌中时，那座地下城只有**1层**——不是一个完整的地下城，只是一个"地下城的雏形"。

这是所有地下城中绝无仅有的情况。
- 普通地下城的最小规模是6层，低于6层的城不会诞生意识
- 但主角的分身城在1层时就拥有了完整的人类意识（因为意识来自灵魂分身，而非核心自然诞生）

### 4.2 开盲盒机制

主角无法控制分身地下城的命运——他被困在"混沌盲盒"中：

- 他不知道自己在地下城混沌中的位置（城市的哪个方向、多远）
- 他不知道自己与近期降临的城是"邻居"还是"孤城"
- 他不知道自己地下城周围的混沌环境——是密集区还是荒漠
- 他不知道自己什么时候会自动现世——只知道"感觉差不多了"的时候

**关于"邻居"**：地下城在混沌中孕育时，相距近的城会产生相互影响。两个城的孕育过程会互相加速，但也会造成空间的不稳定——可能出现入口重叠或连通的情况。

### 4.3 第一次降临与自动现世

主角分身城的第一次降临是一个特殊过程：

- 前期分身城刚降临阶段，主角本身没有搞清发生了什么——甚至不知道自己多了个地下城。分身的第一人称感知无法正常传达给主体，主体的精神力不足以无时无刻支援分身
- 分身城尚未真正"现世"之前，主角处于"我好像背后有更大的事没搞清"的不在线状态（通过大事件触发才有"我竟然是个城"的觉醒）
- 分身城以"最低存在感"降临到蓝星现实世界的角落

### 4.4 入口固定

分身城的入口和所有普通地下城一样——**固定在蓝星某处，不可移动**。

这意味着：
- 主角无法将地下城"带在身边"
- 如果需要亲自进入自己的地下城，必须前往入口位置
- 入口被发现的时间取决于降临位置——如果降在无人区，可能在很长一段时间内只有主角自己知道入口在哪

### 4.5 入口暂时关闭

| 场景 | 效果 |
|------|------|
| **主体亲自进城** | 入口可暂时关闭（代价：消耗源质），防止其他人在主体入城时闯入 |
| **主体离开后** | 入口重新开放，恢复可进入状态 |

这种"暂时关闭"功能是有代价的——关闭期间消耗大量源质，且无法在战斗中突然关闭。

### 4.6 限制条件表

| 限制项 | 说明 |
|--------|------|
| **分身无法离线操控** | 主体必须保持意识清醒才能主动调控地下城。主体昏迷/睡眠时分身自动托管 |
| **分身心智上限** | 分身只能处理与地下城经营直接相关的事——无法通过分身边看世界新闻边管理城 |
| **距离无影响但精力有上限** | 离地下城多远都能操控，但切换视角需要精力，连续高强度的多线操作会导致精神疲劳 |
| **分身无法主动控制职业面板** | 双面板之间不存在"分身直接帮主体升级"的途径——两者成长体系独立 |
| **只有地下城本体可以进入** | 分身不能离开地下城。主体可以进入分身的地下城，但每次进入消耗源质，且进入期间职业者面板和操控面板不可同时开启 |

## 五、源质转经验（主角特有转化）

### 5.1 核心机制

主角拥有一个普通地下城/普通职业者都没有的**越界能力**：他可以将地下城分身积累的**源质**转化为**职业者经验**。

```
源质（地下城运营资源）→ 经验（职业者个人成长资源）
```

这是两套本不相通的体系之间的跨系统转化——世界底层规则中的漏洞型能力。

### 5.2 转化代价

| 维度 | 代价 |
|------|------|
| **转化效率** | 极低：大量源质只能换取少量的经验。源质在转化过程中绝大部分被消耗掉了 |
| **源质枯竭风险** | 转化消耗的是地下城的运营资源。转化过多→无源质维持怪物→城变空→暴露 |
| **冷期** | 转化本身需要源质消耗，源质的再生需要时间 |

### 5.3 使用策略

- 这是一个需要**平衡取舍**的能力——不是"无限经验"的捷径
- 主角不能单纯依赖转化升级——他必须同时探索其他地下城获取正常经验
- 转化是"应急手段"而非"主要升级路径"——应对特殊时期（如急需等级突破、职业者面板遇到瓶颈）

## 六、被攻克=死亡（核心绑定风险）

**核心绑定**是最重要的机制设定之一，也是主角面临的最大风险。

### 6.1 绑定本质

分身不是"寄居"在某个随机地下城中——它是**被绑定**在这个核心上的。分身=核心，核心=分身：

- 分身是这座城核心的**灵魂/意识**
- 核心是分身的**载体/身体**
- 二者是一体的——不可分离

### 6.2 被攻克后果

如果这座地下城被职业者攻克（核心被摧毁）：

```
核心被摧毁→分身（灵魂）随之毁灭→主体当场死亡
```

这不是"损失一个分身"——**就是李冠阳自己的死亡**。

分身毁灭，等价于灵魂的一半被彻底抹除。剩余的一半灵魂无法独立存活——主体的身体也会同步死亡。

### 6.3 核心绑定风险的意义

- **这不是游戏**——不是"大号带小号"或"资源号"的逻辑
- 主角经营这个地下城，就是在保护自己的生命
- 每一支进入他城的队伍，都是潜在的"处刑团"
- 他必须小心翼翼地平衡"吸引职业者进来刷"和"别让他们真的打到核心层"之间的关系
