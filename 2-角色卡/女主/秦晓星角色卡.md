# 秦晓星角色卡

> 女主角色卡。与李冠阳同校不同系，第1章因被追事件一同被吸入地下城，被迫组队。
> 
> 核心定位：不好惹但不蠢，观察力敏锐，。

---

## 一、身份背景

| 项目 | 内容 |
|------|------|
| **姓名** | 秦晓星 |
| **年龄** | 大一学生（约19岁） |
| **就读学校** | 与李冠阳同校，不同系 |
| **家庭背景** | 待填（预留接口：普通家庭/特殊背景待定） |
| **与李冠阳的关系起点** | 见过面但不认识。走廊擦肩级别。李冠阳对她有模糊印象——同校学生 |

### 1.1 当前状态

- 第1章：因得罪了人被追——三人追兵，被迫逃往废弃商场
- 第2章：在地下城战斗中被逼觉醒
- 第3-5章：与李冠阳保持联络，独立适应觉醒者身份
- 性格决定她不会主动求助，但也不排斥合作

---

## 二、性格特征

### 2.1 核心性格

| 特质 | 说明 |
|------|------|
| **不好惹** | 被追的原因就是她惹了不该惹的人，或者惹了人但不低头。不怂，敢刚 |
| **不蠢** | 知道什么时候该跑。不会因为"面子"死撑。跑进废弃商场说明她有判断力——打不过先撤 |
| **观察力强** | 第3章起她第一个发现"这座城不对劲"。她能察觉到世界的变化比别人敏感 |
| **不圣母** | 不会因为"应该救人"而冒险。帮不帮看值不值，救不救看能不能 |
| **独立** | 不会把希望寄托在别人身上。被迫组队≠依赖对方 |

### 2.2 行为模式

- **先评估再行动**：进任何新环境先判断"这里有没有危险"
- **克制表达**：不多话。必要时才开口
- **警觉性高**：废弃商场里她第一个发现石门不对。战斗时反应快
- **不强出头但不退缩**：不主动当队长，但需要她打的时候不手软

### 2.3 当前心理状态

- 第1章：被追时的紧张 → 被吸入地下城后的警惕与戒备
- 第2章：觉醒后的混乱期——"我变成了什么？"
- 第3-5章：逐渐适应，开始独立处理觉醒者身份带来的变化

---

## 三、第1章行为锚定

| 事件 | 行为 | 心理状态 |
|:----:|------|---------|
| 被追 | 跑——不回头，不废话 | 被追的紧张感 |
| 跑进废弃商场 | 选入口隐蔽，判断"这里能甩掉他们" | 快速决策，不犹豫 |
| 看到石门 | 停下、警戒、判断"这不对" | 警觉阈值高 |
| 石门自开 | 后退一步，但没有失去镇定 | 警惕但不慌乱 |
| 被吸入 | 第一反应——站稳、四望、找出口 | 适应力强，不浪费时间去恐慌 |
| 在城中 | 不说话，观察周围 | 零信任阶段 |
| 与李冠阳相遇 | 打量对方——"你也是被卷进来的？" | 戒备但愿意交换基础信息 |

### 3.1 与李冠阳的关系起点

- **零信任**：被迫同行，不是主动选择
- **第2章末**：才确认"对方也不是普通人"
- **不会主动套近乎**，但也不排斥基本合作

## 四、觉醒设定

### 4.1 职业信息

| 项目 | 内容 |
|:----:|------|
| **觉醒职业** | 待定（战斗型推荐：刺客/剑士/弓手） |
| **职业定位** | 实战型——战斗中快速成长 |
| **觉醒时机** | 第2章——在地下城战斗中被逼觉醒 |
| **觉醒方式** | 被怪物逼到死角，本能爆发 |

### 4.2 觉醒后定位

- **双线并行**：第3章起她独立处理自己的觉醒者事务
- **与李冠阳不绑定**：有自己的战斗线、成长线
- **第5章独立事件**：遇到自己的第一个小型战斗/事件——与男主不同步
- **后期定位**：待定（高成长性战斗型/辅助型/特殊型）

## 五、关键人际关系

### 5.1 与李冠阳的关系发展

| 阶段 | 章节范围 | 关系状态 |
|:---:|:-------:|---------|
| **零信任** | 第1章 | 被迫同行，不主动对话 |
| **被迫组队** | 第2章 | 一起战斗/觉醒，建立基础信任 |
| **独立联络** | 第3-5章 | 交换了联系方式，各自处理觉醒事务，偶尔交流信息 |
| **合作建立** | 后续章节 | 外部事件推动下逐渐建立真正的合作关系 |

> ⚠️ 感情线发展原则：**外部事件推动，不告白。** 关系进展靠共同经历的事件自然推进，不依赖直白的情感表达。

### 5.2 追兵三人组

| 关系 | 说明 |
|:---:|------|
| **被追的源头** | 第1章被追的原因——校园冲突/得罪了人 |
| **一同被吸入** | 三人也进了地下城，但在第2章分散 |
| **后续可能** | 三人可能在后续章节再次出现——作为冲突源或剧情回调 |
| **关系定性** | 对立关系。但不一定是"死敌"——可能是可化解冲突 |

### 5.3 与李冠阳的社会关系差异

| 对比项 | 秦晓星 | 李冠阳 |
|:------:|:------:|:------:|
| **社交圈** | 可能更广（有矛盾的人多=认识的人多） | 窄（孤儿院出身，社会关系简单） |
| **冲突来源** | 人际冲突（被特定的人追） | 被动卷入（被地下城吸入） |
| **学校存在感** | 有一定知名度（被追→有人知道她） | 低存在感（没人注意） |
