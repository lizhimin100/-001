### 1.1 现世身份

| 项目 | 内容 |
|------|------|
| **姓名** | 李冠阳 |
| **年龄** | 大一学生（约19-20岁） |
| **出身** | 孤儿院长大，无父无母，无家族背景 |
| **社会关系** | 社会关系简单——有一起从孤儿院出来的朋友，但不多。人际关系圈窄 |

### 1.2 现状

- 刚觉醒没多久，处于"搞不清自己身上发生了什么"的阶段
- 职业者面板显示他是法师，但还有一个奇怪的核心操控面板
- 隐约感觉自己"背后还有更大的事没搞清楚"
- 分身城尚未真正"现世"——那个1层的雏形还在混沌中飘

### 1.3 转世者身份（暗线）

| 项目 | 内容 |
|------|------|
| **前世的认知锚点** | 记得某些"前世"的碎片——具体内容待定，仅作为后期剧情线接口 |
| **当前的记忆状态** | 转世后记忆不完整，只有碎片性的模糊残留。超凡觉醒后，部分碎片开始被扰动 |
| **转世与金手指的关系** | 双面板结构的根源：转世时灵魂与世界底层规则异常交互，灵魂被分裂成主体（现实）和分身（地下城核心） |

> ⚠️ 转世者身份是**暗线**，开局阶段仅通过极少的记忆碎片暗示，不在前5章直接揭示完整背景。转世的具体细节在写作过程中逐步填充。

## 二、职业信息

### 2.1 明面职业：法师

| 项目 | 内容 |
|------|------|
| **职业类型** | 战斗系 - 远程输出 |
| **职业定位** | 常见职业，不会引人注目 |
| **当前等级** | 一转初期（超凡出现约3周，全球觉醒者普遍处于此阶段） |
| **技能树** | 法师基础技能树：火球术/冰刺/魔法护盾等（具体技能列表待写作时细化） |
| **升级方式** | 和其他职业者一样——进入地下城战斗，击杀怪物获取经验 |
| **作战风格** | 谨慎型——不会因为自己有"后手"就浪战。能远程风筝绝不近身，能组队绝不单走 |

### 2.2 法师身份的掩护价值

- 法师常见 → 不会因为职业稀有被重点关注
- 合理输出位 → 可以名正言顺和其他觉醒者组队
- 不引人注目 → 低调发育，不容易被人盯上

## 三、暗面：地下城核心分身

### 3.1 分身是什么

- 分身不是"第二人格"或"宠物"
- 分身是**他灵魂的另一半**——位于异空间混沌中一座正在孕育的地下城核心中
- 分身 = 核心，核心 = 分身——二者绑定，不可分割
- 分身有独立的核心操控面板，可以管理地下城的一切

> 分身如何运作、操控内容、限制条件等完整说明见 [`04_金手指系统.md`](../2-设定/创作前设定/04_金手指系统.md)

### 3.2 分身的当前状态

- 所在的地下城仅1层——一个"地下城的雏形"
- 尚未"现世"（未真正降临蓝星）
- 分身城还在异空间混沌中飘浮
- 主角每天能感知到分身城的状态，但无法完全掌控其命运

## 四、性格特征

### 4.1 核心性格

| 特质 | 说明 |
|------|------|
| **谨慎** | 不会因为有了金手指就浪。在任何决策中优先考虑风险——"这个坑我能不能活着出来"永远排在"这个坑有没有好东西"前面 |
| **务实** | 不做没把握的事，不冒没必要的险。升级慢一点没关系，死了就什么都没有了 |
| **有底线** | 不是圣母，但也不是冷血。利益优先，但不会为了利益伤害无关的人 |
| **适应力** | 孤儿院的成长经历让他习惯"世界随时会变"。对超凡世界没有恐惧期，更多是"那现在我该怎么活"的实用主义思考 |
| **多疑** | 不会轻易相信别人。尤其是在发现自己有两套面板后，对"这个世界到底是什么"更警惕 |

### 4.2 行为模式

- **先观察再行动**：进任何新环境先观察，找出口、找退路、评估风险
- **不主动出头**：在队伍里保持低调，输出够用就行，不会展示全部实力
- **信息守财奴**：他知道的信息，不会轻易告诉别人。信息差是生存资源
- **底线守护者**：对自己人（认定可以信任的人）会护短，但对陌生人保持距离

### 4.3 当前心理状态

- 刚觉醒——混乱期："我到底怎么了？这个面板是什么？那个分身又是什么？！"
- 金手指带来的危机感多于兴奋感——"如果这座城被攻克我会死"这一认知让他很不安
- 同时又有强烈的生存渴望："既然我有了两条命（或者说一条命两个形态），那我就要活得比任何人都久"

## 五、两项面板来源简述

### 5.1 职业者面板来源

和其他职业者一样——因转世后的人类肉身受到超凡能量浸染而觉醒。

但觉醒的时机与转世碎片有关联——他在转世前就对这个世界的力量体系有模糊记忆，超凡降临后他的身体更快地与之共鸣。

### 5.2 核心操控面板来源

来自灵魂的另一半——那座在异空间混沌中的地下城核心。

当他的灵魂被分裂时，分身同时被嵌入了地下城核心的"胚胎"中。核心面板是分身作为核心意识自然拥有的管理界面。
