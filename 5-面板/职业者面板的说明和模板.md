# 职业者面板的说明和模板

> 对标样本书：《超神机械师》系统面板风格——键值列表式，`——————` 做分隔线，无边框字符，简洁清晰

## 一、定义

职业者面板是觉醒者（职业者）在觉醒时获得的标准系统面板，由世界底层规则自动生成并绑定至觉醒者个人。每个觉醒者拥有且仅拥有一个职业者面板。

> **主角特殊性**：李冠阳拥有双面板（职业者面板 + 地下城面板），详见 `../4-设定/创作前设定/04_金手指系统.md`

## 三、各区块详细说明

### 3.1 基础信息区

展示角色的核心生存数据与定位：

| 字段 | 说明 |
|------|------|
| **姓名** | 绑定灵魂，不可更改 |
| **职业** | 觉醒时由规则自动匹配决定，不可更改——不存在"转职"或"兼职" |
| **等级** | 一转（1-99级）→ 二转（100-199级）→ 三转（200-299级）→ 以此类推 |
| **经验值** | 通过击杀魔物/完成任务获得，满额后自动升级 |
| **生命值** | 受体质属性影响，归零即死亡 |
| **能量值** | 施放技能消耗的资源，受主属性影响，且根据主属性细分为魔力值、斗气值、气血值等） |

> **等级/转位框架详见** `../4-设定/创作前设定/02_力量体系.md`

### 3.2 属性区

以普通成年人平均值 10 为基准，觉醒时根据职业特性自动偏移：

| 属性 | 说明 | 法师倾向 | 剑士倾向 |
|------|------|---------|---------|
| **力量** | 物理攻击能力 | 偏低 | 偏高 |
| **敏捷** | 速度/反应/闪避 | 中等 | 高 |
| **体质** | 生命值/防御/耐力 | 偏低 | 中等 |
| **智力** | 法术强度/技能效果 | **核心属性** | 偏低 |
| **精神** | 能量值上限/恢复速度 | **核心属性** | 中等 |
| **感知** | 察觉隐藏/探测范围 | 偏高 | 中等 |
| **魅力** | 社交/交涉/统率 | 因人而异 | 因人而异 |
| **幸运** | 隐藏属性，面板不显示具体数值（显示"?"） | — | — |

### 3.3 技能区

```
【技能】
已习得
  · 技能名称·Lv.X（主动/被动/特殊）──── 说明
  · 技能名称·Lv.X（主动/被动/特殊）──── 说明
未习得
  · 技能名称 ──── 消耗XX点经验可习得
  · 技能名称 ──── 消耗XX点经验可习得
```

**规则**：
- 觉醒时所有技能为灰色（未习得），有足够经验值后可主动习得
- 已习得技能可通过持续使用或投入更多经验提升等级
- 技能栏不设数量上限，但面板只显示当前阶位可解锁的技能
- 每位觉醒者的技能树有共性（同职业可习得相同技能），也有差异性（隐藏的或觉醒者自身才能习得的）

### 3.4 专长区

专长代表觉醒者超越基础技能体系的**特殊天赋或特殊成就**：

| 类型 | 示例 | 获取方式 |
|------|------|---------|
| 种族专长 | 无（蓝星人类均无种族专长） | — |
| 职业专长 | 法术专精：火系法术伤害+5% | 职业等级达到特定条件解锁 |
| 成就专长 | 初战告捷：首次击杀精英魔物 | 达成特定事迹触发 |
| 隐藏专长 | ？？？ | 满足隐藏条件自动解锁 |

> 专长区多数觉醒者为空——需要特定条件才能解锁。大部分职业者终其一生解锁不超过2-3个专长。

### 3.5 功能区（通用功能，所有职业者均有）

扫描功能为**所有职业者的标配功能**，位于面板底部功能栏：

```
【功能】
[扫描]  [日志]  [任务]
```

**扫描规则**：
- 只能扫描**视线范围内**的目标（具体距离上限面板不标注）
- 扫描结果仅显示目标是否职业者：非职业者→「目标：普通人类 / 职业：无」；职业者→「目标：XXX / 职业：XXX」
- **无法**扫描出目标的具体等级、技能、属性——只能判断是否为职业者以及职业类别

**其他通用功能**：

| 功能 | 说明 |
|------|------|
| **日志** | 默认可用——记录战斗数据、击杀信息和关键事件的时间戳 |
| **任务** | 默认可用——职业者面板有任务引导功能，记录待办事项和目标进度 |

## 五、关键约束

| 约束 | 说明 |
|------|------|
| **唯一性** | 每人只有一个职业者面板，不可转移、不可复制、不可伪造 |
| **不可隐藏** | 面板存在于意识层面，无法被物理手段检测或屏蔽，但他人无法直接查看你的面板内容 |
| **共享限制** | 面板内容仅本人可见，截图/口述可分享，但无法通过系统进行面板间信息互通 |
| **觉醒后固定** | 职业在觉醒时由规则自动匹配决定，不可更改——不存在"转职"或"兼职" |
| **扫描范围模糊** | 面板不显示扫描功能的具体距离上限——需要使用者自行摸索边界 |
| **无内置教程** | 觉醒后面板不会弹出使用说明——所有功能需要使用者自己探索发现 |

## 六、模板使用说明

1. 创建角色卡时，直接复制模板填充即可
2. 剧情中首次展示面板时，使用§二的全貌展示格式，用 `——————` 包裹
3. 后续更新时只展示变化项（技能升级、属性变化等），不重复全面板
4. 扫描功能在正文中的展示格式保持统一：`[扫描结果：目标——XXX / 职业——XXX]`
