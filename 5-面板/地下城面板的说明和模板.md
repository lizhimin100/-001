# 地下城面板的说明和模板

> **对标样本书**：《超神机械师》系统面板风格——键值列表式，`——————` 做分隔线，无装饰字符

## 一、定义

地下城面板，全称**地下城核心状态面板**，是李冠阳**独有的暗面系统面板**——通过灵魂分身嵌入地下城核心而获得的操控界面。

> **独有性**：世界上唯一一个拥有地下城面板的人类。普通地下城核心的意识只有本能感知（能感知入侵者、层内状态），但没有"系统面板"这种信息呈现方式。

---

## 二、面板样式说明

### 2.1 基础格式

面板以列表格式展示，每条数据一行，结构统一：

```
——————
[字段名]值
[字段名]值
——————
```

- 字段名和值之间无冒号
- 用 `——————` 包裹起止
- 区块之间留空行分隔

### 2.2 首次完整展示（参考样式）

以下是主角**首次意识同步**时看到的地下城面板：

```
——————
【核心状态】
姓名：未命名
形态：地下城
核心品质：Lv.1
当前阶层：1/1
状态：未现世
源质：0
源质产出：0/天→1/天
核心完整性：100%
阶层信息：[第一层]
天赋：无
宝物：无
地图：未命名的地下城地图（可切换至其他地下城地图）
日志：核心意识苏醒！以当天为日志第一天。
——————
```

## 三、面板字段说明

### 3.1 核心状态字段

| 字段 | 说明 |
|------|------|
| **姓名** | 初始为"未命名"。命名后固定显示为城名——世界底层规则会将该名称注册为正式城名 |
| **形态** | 固定为"地下城"。身份标识，不可更改 |
| **核心品质** | 地下城核心的等级。LV.1初始，消耗大量源质可提升。品质越高→解锁更多操控权限和功能 |
| **当前层数** | 当前已有层数/最大可解锁层数。初始为1/7。消耗源质可增加新层 |
| **现世状态** | 未现世/已现世。未现世时地下城不暴露于地表世界 |
| **源质** | 当前累计的源质总量。来源：击杀入侵者→吸收血肉能量转化 |
| **源质产出** | 自然源质恢复速率（X/天）。随核心品质提升而增加 |
| **核心完整性** | 核心的稳定度。受创后下降→0%则核心被摧毁→分身随灭→主体死亡 |

### 3.2 层信息字段

| 字段 | 说明 |
|------|------|
| **类型** | 该层的空间布局风格：迷宫 / 大厅 / 通道 / 矿脉 / 特殊 |
| **状态** | 稳定 / 受创 / 刷新中 / 锁定 |
| **魔物** | 该层当前存在的魔物列表（类型+数量） |
| **入侵者** | 该层内是否有入侵者。有则显示人数 |
| **出口** | 出口位置及开启状态 |
```

---

## 四、正文呈现规则

1. 首次意识同步时展示上述完整面板
2. 后续只展示变化项（如层数变化、源质增减、魔物列表更新）
3. 战斗中只弹简讯格式，不打全面板：`[击杀骷髅兵×1，源质+5]`
