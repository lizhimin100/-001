# 第2章 觉醒

窸窣声从黑暗里钻出来，像什么东西拖着骨头在石地上挪。

李冠阳第一反应不是看人，而是看那块新冒出来的面板。

透明的蓝色光晕在视野里缓缓铺开，一行行信息浮了出来：

```
——————
姓名：李冠阳
种族：碳基人类（黄）
模板：凡俗生命→职业者
等级：Lv.0→Lv.1（位阶：D）
经验：0/100
主职业：无→法师
副职业：无

个人属性：力量8、敏捷10、耐力9、智力14、神秘11、魅力7、幸运？
自由属性点：0

生命值：90/90
体力值：135/135
魔力值：140/140

【评价：垃圾战斗力，扔在职业者里面纯纯战五渣，随便一只精锐哥布林都能把你揍趴下】

专长：无

技能：无

潜能点：0
职业知识树：未开启
影响力：无
传说度：0
装备：无
——————
```

李冠阳盯着那个【评价】看了两秒，嘴角抽了一下。

垃圾战斗力。战五渣。哥布林都能揍他。

行，够直白。

他目光往下扫，智力14，比平均值高了快一半。

耐力9、力量8——法师的正常偏向。主职业从"无"变成了"法师"，说明觉醒就是在刚才那一瞬间完成的。

可技能是空的。

他意念一动，像是在心里点了一下"技能"标签。面板立刻展开，弹出一列灰掉的条目：

```
【技能库】
[火球术] ──── 消耗10点经验
[法师护甲] ── 消耗10点经验
[照明术] ──── 消耗10点经验
[冰锥术] ──── 消耗10点经验
```

每个后面都跟着一行小字：经验不足。

他看了一眼经验条：10。

刚好够点亮一个。

还没来得及选，脑子深处那块空了三周的旧面板猛地一震。

像有人把他整个人往下一拽。

李冠阳眼前一黑，再睁开时，视野已经不在岩廊里了。

他站在一间空旷的石室中央。

石壁是天然的，粗糙、潮冷，像刚从山腹里剥出来。

石室正中央悬着一块暗红色的晶体，拳头大小，表面不平，像心脏一样一下下明灭。每一次闪烁，他都能感觉到自己的胸腔跟着震一下。

那是他的另一部分。

不是比喻。

就是字面意思。

李冠阳站在那里，他不知道自己为什么知道。

他脑子里莫名其妙就知道了这件事：那块暗红晶体，就是他。

或者说，是他的分身，是他灵魂切出去的另一部分，

他下意识低头看，那块意识里铺开的旧面板已然截然不同：

```
——————
【核心状态】
姓名：未命名
形态：地下城
核心品质：Lv.1
当前阶层：1/1
状态：未现世
源质：0
源质产出：0/天→1/天
核心完整性：100%
阶层信息：[第一层]
天赋：无
宝物：无
地图：未命名的地下城地图（可切换至其他地下城地图）
日志：核心意识苏醒！以当天为日志第一天。
——————
```

他盯着"未命名"那三个字看了两秒，又扫到"可切换至其他地下城地图"这一条。

其他地下城？

这东西难道还能连到别的地下去？

还没等他细想，视野里又弹出一个选项——[检测到其他地下城！当前地图可切换]。

李冠阳愣了半秒，还是点了进去。
一幅粗糙的立体结构图在意识里铺开。

七层。

从上到下，层层叠叠，像一座被削开的山体。

第1层最亮，且能看到有几十个散落的光点零散分布在其中，有几个在代表他小绿点的位置附近的光点正在靠近。

他皱了皱眉。

地下城？这是什么？

不知道。

反正不像游戏界面里那种好心的自动说明。它什么都不解释，只把东西扔到他眼前，让他自己去猜。

黑暗深处又响了一声。

这次更近。

岩廊里，追兵头目压低声音："都别出声。"

只是等了好长一段时间都没有动静。

等了好几分钟的一名男人，烦躁明显压过了恐惧。

他大步朝秦晓星走过去："不管这是哪儿，先把你收拾了再说。"

他伸手去抓她。

就在他的手即将碰到秦晓星肩膀的那一刻——

黑暗深处传来一声绷弦响。

噗。

一支骨箭从阴影中射出，钉进了暴躁男的右肩。

他整个人被冲击力带得撞向岩壁，血当场顺着衣服往下淌。他惨叫一声，捂着肩膀滑坐下去，脸色瞬间煞白。

秦晓星惊叫着往后退了几步。

追兵头目脸色大变，拔刀挡在前面："什么东西？！"

第三个人也慌了，左右张望，手在发抖。

李冠阳趁乱闪身退入阴影——不是躲，是找位置。

他快速翻看面板。活着才有一切。先点亮护甲。

他意念点向【法师护甲】。

```
[技能习得：法师护甲·Lv.1]
体表覆盖微光层，提供基础物理防护
```

一层极淡的白光从他的皮肤表面掠过去，像水面浮了一层油膜。

追兵头目已经彻底慌了。他看了一眼肩膀被射穿的暴躁男，又看了一眼黑暗深处，咬牙做了决定："跑！"

他拖着受伤的暴躁男，招呼第三个人，转身就往通道另一头跑。脚步声很快被黑暗吞没。

岩廊重新安静下来。

但那种安静只持续了几秒。

三只惨白的骷髅从阴影中缓缓走出。不是完整的人形，像拼接起来的骨架，眼眶里空洞洞的。它们没有去追逃跑的追兵，而是朝还留在原地的两个人逼过来。

秦晓星慢慢后退，一直退到李冠阳附近。

她看到了他手上那层微光。

这个人不一样。至少不是敌人。

---

三只骷髅越逼越近。

李冠阳后退中猛地想起那张地图——那些光点。

他迅速点开地图：上面有三个光点正在从他们前方的位置缓慢靠近，方向和骷髅逼近的方向完全一致。

他心里一紧。

难道这张地图……就是这座地下城的结构图？

他试着用意念点中其中一个光点。

信息弹出：

```
[骷髅战士·Lv.1]
[弱点：颈骨]
```

确认了。

地图上的光点就是这些骷髅的位置。

他把秦晓星往后一推："往左，那边有岔路！"

"你怎么知道那边……"

"信我！"

秦晓星咬了咬牙，没再问，跟着他往左边退。

窄通道只够一人通过，骷髅只能一只一只挤进来。

第一只踏入。

骨刀举起，劈下来。

李冠阳硬着头皮顶上去——法师护甲还在。骨刀砸在他肩膀上，闷得他整条胳膊发麻，但没穿过去。他没退，双手死死扣住那截骨脖子，冲秦晓星喊了一声：

"砸它脖子！"

秦晓星半秒都没犹豫。她捡起地上的石块，对准颈骨最细的地方猛砸下去。

一下。

两下。

骨架发出一声脆响，颈骨断裂，骷髅头滚落在地。整个骨架散了。

```
[击杀骷髅兵×1，经验+10]
```

经验变成20。

技能栏里，【火球术】从灰色变成了半透明——可习得了。

他点了习得。

---

第二只骷髅已经从窄通道挤了进来。

李冠阳抬起右手。

掌心里先是一热，接着一团橘红色的火猛地窜了出来，歪歪扭扭的，像刚学会走路的猫。但它确实飞了出去——砸在骷髅胸口，轰地炸开一片火星。

骨架被炸退了两步，胸骨碎裂，摇摇晃晃。

秦晓星瞪大了眼，嘴巴微张。

显然没想到他还能放火。

"愣什么，跑！"李冠阳喊。

他再次打开地图——光点从三个变成了八个。更多的骷髅涌来了。但从第1层东北角的方向，他看到了一条特殊的标记：不像通道，更像一条裂开的细缝。

出口。

他现在已经能确认了——这张地图就是他现在待的这座地下城的结构图。

他拉起秦晓星："跑！后面通道，往右！"

她一愣："你怎么知道那边——"

"信我。"

她从牙缝里挤出两个字："你最好是对的。"

两人转身狂奔。

旧面板上在这时弹出一行字：

```
[现世倒计时：约72小时]
```

黑暗中，脚步声从四面八方涌来。
