# 1-5章结构大纲（新版）

> **黄金前5章** 三拨人同时吸入 → 被迫组队 → 双线并行
>
> 基于以下约束条件：信息混沌、前20章找不到其他地下城、入侵者主动找上门、法师线以现实活动为主、核心经营是唯一战场。

---

## 第1章《门》

**场景1-1：梦与空白面板（350字）**
李冠阳又梦到了那个地方。黑暗、石壁、一条发光的槽。醒来后凌晨4:17。这已是三周来的第十几次。他点开意识里那块透明屏幕——空空荡荡，一个字都没出现过。三周了。辅导员在群里催住宿费，他没回。卡里剩41块，还要撑一周。

**场景1-2：日常与猜测（400字）**
上课。舍友聊“学校后面旧商业区封了，说是地基异常”。朋友圈有人发“真有人消失了”，评论区半信半疑。李冠阳放下手机，脑子里那根弦绷紧了。三周前开始做梦，三周前出现空白面板，三周前网上开始出现“超凡”“异空间”的帖子。时间线对得上。但他没有证据。

**场景1-3：目睹与跟随（500字）**
下午回宿舍，路过旧商业区外围。李冠阳看到前方有个女生往废弃商场方向跑——脚步急，回头看了一眼，脸色不对。他认出来了：同校，不同系，见过几次，不知道名字。女生身后跟着三个人，明显在追她。她慌不择路跑进了废弃商场的地下入口。追兵也跟了进去。李冠阳站在路口，犹豫了一下。一个女生被三个人追进废弃建筑——不管怎样，这事不能当没看见。他咬牙跟了上去。

**场景1-4：入口投影与确认（450字）**
他摸进废弃商场，躲在承重柱后。地下入口附近，追兵已经堵住了女生的去路。然后空气像被撕开一道半透明的裂口。那一瞬间，李冠阳脑子里那个空白面板**震了一下**——不是弹出文字，是震动，像手机开了静音在意识深处嗡嗡作响。他盯着裂口，心跳加速。梦里那个空间……和这个好像。石壁、黑暗、发光的槽。世界真的变了。

**场景1-5：吸入（300字）**
追兵领头感觉不对劲，想离开，却不料下一刻在场的所有人都感觉到脚下一空——不是吸力，是区域置换。视野切换。李冠阳脚下变成石地。女生在几米外。追兵三人也在。这里是地下洞穴空间。所有人都进来了。

**场景1-6：黑暗中的异常（250字）**
追兵头目开口：“这什么地方？”，众人争吵，就在追兵组一人打算狠狠收拾女生时。黑暗深处，传来一丝动静——像是有什么东西在移动。所有人的话卡在喉咙里。女生秦晓星慢慢后退。窸窣声突然停了。死一般的安静。没有人敢动。

**场景1-7：章末钩子（200字）**
安静中，李冠阳意识里，那块空了三周的屏幕旁边，**多出了一块新面板**。上面浮现一行字：`[职业觉醒条件已满足。]`他愣住。与此同时，黑暗中的动静重新响起——比之前更近。



第二章《觉醒》梗概（最终调整版）
场景2-1：觉醒与地下城核心苏醒（400字）
黑暗中死一般的安静。李冠阳屏住呼吸，意识落在那块新面板上——[职业：法师]。技能栏全是灰色——[火球术·未习得]、[法师护甲·未习得]……每个都需要10点经验。经验一栏显示：[经验：10]。只有10点，刚好够点亮其中一个。还没来得及细看，脑子深处“轰”的一声。那块空了三周的旧面板猛地展开。意识被拽了进去——空旷的大厅，石壁，正中央悬浮着一块暗红色的晶体，拳头大小，像心脏一样有节奏地明灭。他能感觉到石壁的冰冷，能感觉到大厅的寂静，能感觉到那块晶体在跳动。那是他的心跳。那是他的另一部分。一个成为了地下城的分身。旧面板上浮现文字：[地下城核心已苏醒] [当前层数：一层] [状态：未现世]。

场景2-2：骨箭与法师护甲（350字）
追兵头目压低声音：“都别出声。”暴躁男等了几分钟，不耐烦了，大步走向秦晓星：“先把你收拾了再说。”伸手去抓她。黑暗深处传来一声绷弦。骨箭射中暴躁男肩膀，钉入墙体。他惨叫着倒地。秦晓星惊叫着后退。追兵头目拔刀挡在前面：“什么东西？！”李冠阳趁乱闪身退入阴影——不是躲，是找位置。他快速翻看面板，活着才有一切，所以先点亮护甲。消耗10点经验，[法师护甲]亮起。一层微光覆在皮肤上。秦晓星瞥见了他手上的光，愣了一下。

场景2-3：追兵逃跑与骷髅逼近（250字）
追兵头目脸色大变：“跑！”他带着剩下的人转身逃往通道另一头。三只骷髅兵从阴影走出，没有去追逃兵，而是朝李冠阳和秦晓星逼来。秦晓星慢慢后退，退向李冠阳的方向——她注意到他手上那层微光。这个人不一样，至少不是敌人。

场景2-4：联手击杀（400字）
李冠阳点开地下城面板上的光点——[骷髅战士·Lv1][弱点：颈骨]，原来光点是地下城怪物以及介绍啊。他把秦晓星往后一推：“往左，那边有岔路。”两人退入窄通道，骷髅只能一只一只进来。第一只踏入，骨刀举起。李冠阳冲上去，用激活护甲的身体硬扛住骷髅的冲击，双手死死扳住骷髅头。“砸脖子！”他喊道。秦晓星咬牙，举起石头对准颈骨猛砸。一下，两下。李冠阳趁机脱身，最后一脚踹断骷髅头。骷髅散架。面板弹出：[经验+10]，经验变成10点。技能栏里，[火球术]从灰色变成半透明——[可习得，消耗经验：10]。他点了习得，经验归零。

场景2-5：火球术展示与黑暗（300字）
远处骷髅弓手拉弓——射中符文灯。洞穴陷入黑暗。黑暗中一只骷髅从侧面扑来。李冠阳来不及想，右手本能一推——一团拳头大的火球歪歪扭扭飞出，砸在骷髅胸口，炸开一片火星。骷髅踉跄后退。火球术，成了。秦晓星瞪大了眼，嘴巴微张。李冠阳立刻打开地图：光点从六个变成了十二个。他拉起秦晓星：“跑！后面通道，往右！”她一愣：“你怎么知道那边……”“信我。”秦晓星咬了咬牙，跟上了他的脚步。旧面板上多了一行字：[现世倒计时：约72小时]。黑暗中，脚步声从四面八方涌来。

## 第3章《双面板》——场景序列

### 核心目标
- 后遗症：李冠阳发现灰色「心核」
- 第一次连接——感知到另一座城
- 秦晓星开始独立适应觉醒者身份
- 两人保持联络
- 章末提示：降临倒计时开始

### 场景3-1：后遗症（500字）
- 李冠阳回宿舍。浑身痛
- 查看面板——发现多了一个灰色的「心核」
- 心核下方一行小字：「已绑定[未命名地下城]」
- **叙事功能**：暗示他除了法师身份之外还有另一个维度的能力

### 场景3-2：试探（500字）
- 用意念点开心核
- 第一次加载缓慢——像是系统在同步什么
- 加载完成后，意识中出现**第二个视角**
- 不是画面——是深度感知——"在地下城深处，有一个核心在等待他"
- **叙事功能**：让读者和主角同时探索双面板的边界

### 场景3-3：第一次连接（600字）
- 晚上深度连接心核
- 感知到地下城内部——空间大小、墙壁质地、空气中的能量
- 用意念移动某块石头——**石头动了**
- 吓了一跳，断开连接
- **叙事功能**：展示核心分身的能力——不是第二人格，是"第三只手"

### 场景3-4：秦晓星线（400字）
- 她在调整。查到自己的职业信息
- 发现自己觉醒了某个不常见的战斗职业
- 两人保持联络——交换了联系方式
- **叙事功能**：独立线建立——秦晓星有自己的觉醒者路径

### 场景3-5：双线并行（500字）
- 李冠阳开始理解双面板运作方式
- 明面：法师。暗面：城主
- 发现可以同时做两件事——表面上听课，意识一部分沉浸在地下城中
- **代价**：精神消耗加快——第三节课开始头痛
- **叙事功能**：展示双面板的代价机制

### 场景3-6：章末（300字）
- 核心面板推送第一条提示：
- `[降临倒计时开始]`
- 主角不知道这意味着什么
- **章末悬念**："降临"是什么？好的还是坏的？

### 第3章信息释放检查

| 释放了 | 没有释放 |
|--------|---------|
| 双面板存在（法师+心核） | 不展示全部操控权限 |
| 核心分身能力雏形（感知+移动物体） | 不让主角完全理解"分身是什么" |
| 精神消耗机制 | 不让主角知道消耗极限在哪 |
| 秦晓星独立线建立 | 不揭示她的完整职业信息 |
| 降临倒计时 | 不让主角知道降临意味着什么 |

## 第4章《降临》——场景序列

### 核心目标
- 72小时倒计时——主角试图手动干预失败
- 探索自己的城——空的，什么都没有
- 秦晓星在适应觉醒者身份——两人校园碰面
- 降临发生——入口在郊外树林
- 第一次亲手操控入口形状

### 场景4-1：倒计时（500字）
- 72小时倒计时
- 李冠阳不知道这意味着什么
- 试着手动干预降临——失败
- 面板只有一行模糊描述："降临期——地下城与现实世界联通。外人可进入。"
- **叙事功能**：降临不可控——主角被动接受

### 场景4-2：探索自己的城（600字）
- 课余时间通过核心面板感知自己的城
- 空的。1层。什么都没有
- 面积约半个足球场
- 需要源质来造东西——源质为0
- **叙事功能**：源质体系基础概念——"需要先运营才有产出"

### 场景4-3：秦晓星线（400字）
- 她也在适应自己的觉醒
- 两人在校园里碰面
- 对话简短——都在消化"世界变了"
- **关系进展**：从零信任过渡到"可以说话的人"

### 场景4-4：降临发生（500字）
- 72小时后。地下城降临
- 李冠阳在宿舍里感受到一阵震动——来自灵魂深处
- 闭眼感知：入口出现在郊外树林
- 面板弹出：`[降临期：7天]`
- **叙事功能**：降临是有期限的——不是永久开放

### 场景4-5：第一次操控（600字）
- 逃课去郊外。找到入口——地面上黑漆漆的洞口
- 用法师身份进入。亲手摸到自己的城
- 用核心操控调整入口形状——洞口缩小
- 操控10分钟就开始头痛
- **叙事功能**：展示核心操控的实际应用

### 场景4-6：章末（300字）
- 意识到两件事：
  1. 自己的城能操控——和别人不一样
  2. 这个能力不能让人知道
- **章末悬念**：他的城和别人不一样——这是优势，也是危险

### 第4章信息释放检查

| 释放了 | 没有释放 |
|--------|---------|
| 降临机制（72小时倒计时+7天窗口） | 不展示完整的降临规则 |
| 核心操控基础（移动物体、调整入口） | 不让主角知道所有操控项 |
| 源质基础概念（需要经营获取） | 不让主角知道如何高效获取源质 |
| 秦晓星与主角的关系进展 | 不让两人关系过度亲密 |
| 主角意识到自己与众不同 | 不让主角完全理解差异的原因 |

## 第5章《第一波入侵者》——场景序列

### 核心目标
- 降临第3天——三个陌生人进入
- 主角抉择：露脸还是隐藏？
- 用环境驱赶——制造雾气，移动碎石墙
- 秦晓星独立事件——她的第一次战斗
- 章末反思：需要自动化防守体系

### 场景5-1：异动（500字）
- 降临第3天。李冠阳在地下城内层布设基础障碍
- 核心分身感知到入口处有三个人的脚步——不是他的
- 时间差：主角比入侵者更早知道他们进来了
- **叙事功能**：核心感知的预警价值——比入侵者快一步

### 场景5-2：抉择（600字）
- 三个人已经走进第一层
- 主角面临选择：现身？不现身？阻止？
- **第一个战术决策**：不露脸，用环境驱赶
  - 调整通道墙壁——把路引向难走的方向
  - 制造雾气——降低能见度
  - 移动碎石堆——制造障碍
- **叙事功能**：主角第一次在"自己的城"中做出战术决策

### 场景5-3：三人视角（500字）
- 切换到入侵者POV：
  - "这个地方好奇怪，上次进的异空间不是这样"
  - "没有看到怪物……但是好像一直在变？"
  - "刚才那边还有路，现在变成墙了？"
- 不是觉醒者——只是三个好奇的普通人
- 找不到路→恐惧→退出
- **关键**：没有对比基准——只觉得"诡异"，不知道有人在操控

### 场景5-4：秦晓星独立线（400字）
- 她在第5章遇到自己的第一个小型战斗/事件
- 独立线——与李冠阳不同步
- **叙事功能**：展示秦晓星的独立成长路径

### 场景5-5：反思（500字）
- 三人退出后，主角断开核心感知——精神力几乎耗尽
- 意识到：这次是普通人，下次可能是觉醒者
- 不能每次都硬扛——需要"自动化防守体系"
- 用源质生成第一批基础怪物
- 规划陷阱布局
- **叙事功能**：从防守到经营的转折——主角开始认真经营

### 场景5-6：章末（300字）
- 核心面板提示：`[降临期剩余：2天]`
- 他要在2天内让城至少能拦住第二波人
- **章末悬念**：冷却期后——谁能进来？进多少人？他挡不挡得住？

### 第5章信息释放检查

| 释放了 | 没有释放 |
|--------|---------|
| 降临期有外人进入 | 不让入侵者意识到有人在操控 |
| 核心感知预警功能 | 不展示完整感知范围 |
| 基础操控有效性（迷雾、移墙） | 不让主角消耗过量源质 |
| 秦晓星独立事件 | 不揭示她的战斗结果 |
| 自动化防守需求 | 不给"完美的防守方案" |

## 附：新版结构优势对比

| 对比项 | 旧版（单人线） | 新版（三拨人+双线） |
|:------:|:-------------:|:------------------:|
| 开局复杂度 | 单人觉醒→探地图 | 三拨人同时吸入→混乱→分散 |
| 女主引入 | 无（待定） | 第1章出现，第2章觉醒 |
| 感情线起点 | 无 | 零信任→被迫同行 |
| 冲突层次 | 单人 vs 环境 | 三拨人 vs 环境 + 人际张力 |
| 双线并行 | 无 | 第3章起男女主各自独立线 |
| 金手指揭示节奏 | 单人摸索 | 心核贯穿 + 战斗觉醒 |
