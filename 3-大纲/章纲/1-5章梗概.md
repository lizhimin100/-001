# 1-5章结构大纲（新版）

> **黄金前5章** 三拨人同时吸入 → 被迫组队 → 双线并行
>
> 基于以下约束条件：信息混沌、前20章找不到其他地下城、入侵者主动找上门、法师线以现实活动为主、核心经营是唯一战场。

---

## 第1章《门》

**场景1-1：梦与空白面板（350字）**
李冠阳又梦到了那个地方。黑暗、石壁、一个微微发光的圆球。醒来后凌晨4:17。这已是三周来的第n次了。他点开意识里那块透明屏幕——空空荡荡，一个字都没出现过。三周了。辅导员在群里催住宿费，他没回。卡里剩41块，还要撑一周。

**场景1-2：日常与猜测（400字）**
上课。舍友聊"学校后面旧商业区封了，说是地基异常"。朋友圈有人发"真有人消失了"，评论区半信半疑。李冠阳放下手机，脑子里那根弦绷紧了。三周前开始做梦，三周前出现空白面板，三周前网上开始出现"超凡""异空间"的帖子。时间线对得上。但他没有证据。

**场景1-3：目睹与跟随（500字）**
下午回宿舍，路过旧商业区外围。李冠阳看到前方有个女生往废弃商场方向跑——脚步急，回头看了一眼，脸色不对。他认出来了：同校，不同系，见过几次，不知道名字。女生身后跟着三个人，明显在追她。她慌不择路跑进了废弃商场的地下入口。追兵也跟了进去。李冠阳站在路口，犹豫了一下。一个女生被三个人追进废弃建筑——不管怎样，这事不能当没看见。他咬牙跟了上去。

**场景1-4：入口投影与确认（450字）**
他摸进废弃商场，躲在承重柱后。地下入口附近，追兵已经堵住了女生的去路。然后空气像被撕开一道半透明的裂口。那一瞬间，李冠阳脑子里那个空白面板**震了一下**——不是弹出文字，是震动，像手机开了静音在意识深处嗡嗡作响。他盯着裂口，心跳加速。梦里那个秘境空间……和这个好像。石壁、黑暗、发光的槽。世界真的变了。

**场景1-5：吸入（300字）**
追兵领头感觉不对劲，想离开，却不料下一刻在场的所有人都感觉到脚下一空——不是吸力，是区域置换。视野切换。李冠阳脚下变成石质地面。女生在几米外。追兵三人也在。这里是地下城的内部空间。所有人都进来了。

**场景1-6：黑暗中的异常（250字）**
追兵头目开口："这什么地方？"，众人争吵，就在追兵组一人打算狠狠收拾女生时。黑暗深处，传来一丝动静——像是有什么东西在移动。所有人的话卡在喉咙里。女生秦晓星慢慢后退。窸窣声突然停了。死一般的安静。没有人敢动。

**场景1-7：章末钩子（200字）**
安静中，李冠阳意识里，那块空了三周的屏幕旁边，**多出了一块新面板**。上面浮现一行字：`[职业觉醒条件已满足。]`他愣住。与此同时，黑暗中的动静重新响起——比之前更近。

> **设定对齐**：本章的外来城是已现世的**小型城**[7层，参照设定§六]，处于**不稳定期（入口投影阶段）**[参照设定§2.2.1]——入口投影尚未固化为永久入口，但核心能量已活跃到足以触发区域置换。

## 第2章《觉醒》——场景序列

### 核心目标
- 职业觉醒+地下城核心苏醒（双面板确认）
- 发现旧面板上多了一个「地图」选项——**但他不知道这是什么地图**
- 战斗中第一次验证：地图上的光点=骷髅的实际位置
- 验证后确认：原来这张地图就是他们所在的这座地下城的结构图
- 逃生方向依赖地图告知的出口信息——不是运气
- 现世倒计时：约72小时

> **设定对齐**：本章的外来城为**7层活体秘境**，已现世，处于不稳定期（入口投影阶段）。主角被吸入后遭遇的是**第1层**的骷髅兵，位阶为一转1-3级水平。地图显示的是整座城的七层结构，但目前只有第1层有活动光点（其余各层无探测到活动）。

### 场景2-1：觉醒+地图出现——但不知道是什么（500字）
黑暗中死一般的安静。李冠阳屏住呼吸，意识落在那块新面板上——[职业：法师]。技能栏全是灰色——[火球术·未习得]、[法师护甲·未习得]……每个都需要10点经验。经验一栏显示：[经验：10]。只有10点，刚好够点亮其中一个。还没来得及细看，脑子深处"轰"的一声。那块空了三周的旧面板猛地展开。意识被拽了进去——空旷的大厅，石壁，正中央悬浮着一块暗红色的晶体，拳头大小，像心脏一样有节奏地明灭。他能感觉到石壁的冰冷，能感觉到大厅的寂静，能感觉到那块晶体在跳动。那是他的心跳。那是他的另一部分。一个成为了地下城的核心。旧面板上浮现文字：[地下城核心已苏醒] [当前层数：1层] [状态：未现世]。

他本能地继续翻看旧面板——意外发现核心页面旁边多出了一个**「地图」选项**。点开——意识中浮现一幅粗糙的立体结构图。**七层结构**——从第1层到第7层的纵向剖面，通道走向互相连接，十几个散落的光点集中在第1层。**但他不知道这是什么地图**。没有标注地名，没有文字说明。这张图看起来像某座地下城的结构图——但对应的是哪座？不知道。那些光点是什么？不知道。还没来得及细想，意识就被拉回了现实——黑暗中的骨骼摩擦声比刚才更近了。

**叙事功能**：地图第一次出现时是"未解之谜"——主角只知道有这个工具，但不清楚它的用途和对应关系。

### 场景2-2：骨箭与法师护甲（350字）
追兵头目压低声音："都别出声。"暴躁男等了几分钟，不耐烦了，大步走向秦晓星："先把你收拾了再说。"伸手去抓她。黑暗深处传来一声绷弦。骨箭射中暴躁男肩膀，钉入墙体。他惨叫着倒地。秦晓星惊叫着后退。追兵头目拔刀挡在前面："什么东西？！"李冠阳趁乱闪身退入阴影——不是躲，是找位置。他快速翻看面板，活着才有一切，所以先点亮护甲。消耗10点经验，[法师护甲]亮起。一层微光覆在皮肤上。秦晓星瞥见了他手上的光，愣了一下。

### 场景2-3：追兵逃跑与骷髅逼近（250字）
追兵头目脸色大变："跑！"他带着剩下的人转身逃往通道另一头。三只骷髅兵从阴影走出，没有去追逃兵，而是朝李冠阳和秦晓星逼来。秦晓星慢慢后退，退向李冠阳的方向——她注意到他手上那层微光。这个人不一样，至少不是敌人。

### 场景2-4：联手击杀——第一次验证地图（400字）
三只骷髅逼过来。李冠阳后退中猛地想起那张地图——**那些光点……** 他迅速点开地图：上面有三个光点正在从他们前方的位置缓慢靠近，方向和骷髅逼近的方向一致。**他心里一紧——难道这张地图……就是这座城的结构图？** 他试着点中其中一个光点——弹出信息：[骷髅战士·Lv1][弱点：颈骨]。**确认了。** 地图上那些光点就是这些骷髅的位置。他把秦晓星往后一推："往左，那边有岔路。"两人退入窄通道，骷髅只能一只一只进来。第一只踏入，骨刀举起。李冠阳冲上去，用激活护甲的身体硬扛住骷髅的冲击，双手死死扳住骷髅头。"砸脖子！"他喊道。秦晓星咬牙，举起石头对准颈骨猛砸。一下，两下。李冠阳趁机脱身，最后一脚踹断骷髅头。骷髅散架。面板弹出：[经验+10]，经验变成10点。技能栏里，[火球术]从灰色变成半透明——[可习得，消耗经验：10]。他点了习得，经验归零。

**叙事功能**：第一次验证——光点移动方向≈骷髅出现方向，点击光点弹出怪物信息，地图的用途开始浮出水面。

### 场景2-5：火球术展示——验证地图上的出口信息（350字）
远处骷髅弓手拉弓——射中符文灯。秘境空间陷入黑暗。黑暗中一只骷髅从侧面扑来。李冠阳来不及想，右手本能一推——一团拳头大的火球歪歪扭扭飞出，砸在骷髅胸口，炸开一片火星。骷髅踉跄后退。火球术，成了。秦晓星瞪大了眼，嘴巴微张。李冠阳再次打开地图——光点从六个变成了十二个——更多骷髅涌来了。但这次他注意看地图的整体结构：**七层纵向排列**，通道走向清晰——而第1层东北角有一个特殊的标记：**一条通向外的裂隙**。他现在已经能确认了：**这张地图就是他现在待的这座地下城的结构图**。出口信息就在地图上。他拉起秦晓星："跑！后面通道，往右！"她一愣："你怎么知道那边……""信我。"秦晓星咬了咬牙，跟上了他的脚步。旧面板上多了一行字：[现世倒计时：约72小时]。黑暗中，脚步声从四面八方涌来。

**叙事功能**：第二次验证——地图的整体结构=当前地下城的七层结构；出口信息成为逃生决策依据；信息差优势正式建立。

### 第2章信息释放检查

| 释放了 | 没有释放 |
|--------|---------|
| 职业觉醒——李冠阳是法师 | 不展示法师完整技能树 |
| 地下城核心苏醒（暗红晶体、分身感） | 不揭示核心全部功能 |
| 旧面板有「地图」选项——但主角一开始不知道是什么 | 不展示地图的完整精度/更新频率 |
| 【验证1】光点移动方向=骷髅靠近方向（第一次猜测） | 不让主角在测试前就完全理解 |
| 【验证2】点击光点弹出怪物信息（确认） | 不让主角知道跨层探测限制 |
| 【验证3】地图整体结构=当前地下城（出口信息确认） | 不让地图的显示机制完全透明 |
| 地图显示七层结构（但只有第1层有活动光点） | 不让其他层的细节暴露 |
| 法师护甲（物理防护、微光） | 不让护甲数值透明 |
| 火球术（远程、歪扭、音效） | 不让技能熟练度机制暴露 |
| 出口方向来自地图信息（不是运气） | 不让此时揭示地图的其他功能 |
| 现世倒计时：约72小时 | 不让倒计时机制透明 |
| 秦晓星注意到李冠阳有特殊信息 | 不让秦晓星追问到底——有限信任 |

## 第3章《逃生·分身》——场景序列

### 核心目标
- 利用已验证的地图找到逃生路线——信息差优势落地执行
- 逃亡过程中两人第二次联手战斗
- 从外来城中逃脱——两人各有下落
- 高烧中「心核」浮现——意识深处发现灰色核心
- 第一次感知到遥远的另一座城
- 秦晓星短信互动——第一个共同秘密
- 前世记忆碎片提示——他不是蓝星原住民

> **设定对齐**：第2-3章的外来城是**7层活体秘境**，已在不稳定期现世。主角在第1层活动，未涉足其余各层。逃出路线利用的是投影不稳定期产生的空间裂隙（出口），不是正常层间通道。

### 场景3-1：地图与决策（500字）
- 火球术炸开通道后，两人狂奔
- 身后脚步声越来越多——黑暗中的骨骼摩擦声从四面八方涌来
- 秦晓星跑在前面，回头喊"往哪走？"
- 李冠阳一边跑一边快速翻看地图——现在已经确认这就是这座城的结构图
- 地图显示：**七层立体结构**，他们在第1层深处；出口在第1层东北角有一条裂隙（不稳定的投影裂口），通向外部；其余各层有通道连接但无活动光点
- 他锁定方向："跟我来，右边！"
- 秦晓星："你怎么知道？！"
- 他没时间解释——"信我"——她已经听过一次了
- **叙事功能**：地图信息已在前一章完成验证，这一章直接用——信息差优势从"发现"进入到"运用"阶段

### 场景3-2：岔路遭遇战（600字）
- 两人拐进右侧通道——更窄，更暗
- 但刚跑出二十米，前方阴影里三只骷髅兵站起来了
- 前后夹击——退路被追兵堵死，前路有骷髅挡道
- 秦晓星本能侧身、压低重心——没等李冠阳开口，她已经冲出去了
- **第一次真正意义上的双人配合战斗**：
  - 秦晓星正面冲上去——侧身避开骷髅骨刀横扫，一拳砸在骷髅头侧面，震碎下颚骨；另一只骷髅从侧面扑来，她一个低扫绊倒它，膝盖压住胸口，捡起掉落的骨刀扎进颈骨缝隙——干净利落
  - 李冠阳在后面补位——法师护甲挡下追来的骷髅骨箭，反手一发火球术封住来路通道（骷髅被震退，通道塌了一小段，暂时拦住追兵）
- 两人都没说话，但动作已经对上了——一个冲，一个守；一个近，一个远
- **叙事功能**：战斗展示双人配合+"剑士战斗风格"的第一次完整呈现（敏捷、干净、不依赖面板）

### 场景3-3：裂隙出口——最终验证（400字）
- 最后一只骷髅散架后，两人继续往前跑
- 按照地图指示——前方是一条向斜上方延伸的天然裂隙
- 裂隙很窄，只能侧身通过——秦晓星先钻过去，在另一边伸手拉他
- 两人合力搬开出口处的碎石堆——外面透进一丝光
- 挤出去——跌入废弃商场的地下室
- 身后裂隙传来轻微的震动——像秘境入口"收拢"了——通道消失
- 瘫坐在地上，满身伤痕
- 李冠阳心里默默确认：**地图是对的。** 从一开始的困惑、到猜测、到验证、到这一刻——他终于彻底相信了那张地图
- **叙事功能**：出口可验证——地图的完整可靠性在此刻得到最终确认

### 场景3-4：别问（400字）
- 两人瘫坐在废弃商场地下室的灰尘里，喘了很久
- 秦晓星盯着他看了几秒：
  - "你刚才看的是什么？"
  - "面板。"
  - 她沉默了一会儿。她也有面板，但她没有看到地图。那他的面板为什么能看到？
  - 她没有追问——不是不想知道，是知道问了也不会全说
- 各自爬起——秦晓星先走，走时回头看了他一眼，眼神复杂
- **叙事功能**：信息差在两人之间建立——她知道他有自己不知道的东西；有限信任的微张力

### 场景3-5：高烧·心核（600字）
- 李冠阳回宿舍——强撑着爬上床
- 高烧——浑身剧痛，像被车碾过
- 但奇妙的是：高烧没有让意识模糊——反而更清醒了
- 意识深处浮现一块灰色的「心核」
- 不是面板弹窗——是"看到"了自己的意识空间里有一个东西
- 暗沉的灰，表面粗糙——像一颗胚胎
- 他发现自己同时站在两个地方：
  - 宿舍床上，烧得浑身发烫
  - 远方一座漆黑的秘境空间里——最深处的那个心核就是他自己，另一个他
- **关键设定**：他不知道为什么自己会知道——但他就是知道，像天生该明白的东西
- 他"知道"那座城还在混沌中，尚未完成定型降临
- 他"知道"那座城很远——非常远——在南方极远处，方向模糊
- 他"知道"那里什么都没有——空的、荒芜的、原始岩石洞窟
- **叙事功能**：心核≠工具——它是"另一个我"；"天生就知道"的信息释放方式区别于面板式说明

### 场景3-6：章末（300字）
- 手机震了。秦晓星的短信：「你也疼？」
- 李冠阳盯着屏幕看了几秒，回了一个字：「嗯」
- 隔了一会儿，她又发：「你那个……是什么职业？」
- 他：「法师。你呢？」
- 她：「剑士。」
- 沉默。两人有了第一个共同秘密。
- 手机放下，高烧中意识越来越沉
- 脑海中闪过一些不属于这里的碎片——孤儿院、食堂的饭、熬夜打游戏、被"大运"送走时的模糊画面
- 他不是蓝星的原住民。他"回来"了。但怎么回来的？为什么？不知道。
- **章末悬念**：穿越者身份暗示——他不是这个世界的人。

### 第3章信息释放检查

| 释放了 | 没有释放 |
|--------|---------|
| 地图已在第2章完成验证→本章直接用于决策 | 不展示地图的完整精度 |
| 逃生路径有据可查（不是运气） | 不让主角知道地图的更新频率/机制 |
| 双人配合战斗（剑士正面+法师补位封路） | 不让配合过于熟练（仍是第一次） |
| 秦晓星战斗风格（敏捷、干净、不依赖面板） | 不揭示剑士完整技能体系 |
| 心核存在（意识空间中的灰色核心） | 不展示心核全部功能 |
| "天生就知道"的信息获取方式 | 不让主角理解这种直觉的来源 |
| 分身城在南方极远处、方向模糊 | 不给精确定位 |
| 穿越者身份碎片（孤儿院、游戏、"被送走"） | 不让前世记忆完整回归 |
| 秦晓星注意到信息差（他有她不知道的东西） | 不让追问深入——有限信任 |

## 第4章《心核·感知》——场景序列

### 核心目标
- 第一次"跨次元感知"——连接远方的分身城
- 日常操控不耗精神的设定验证（移动碎石、调整通道）
- 创造行为消耗源质的机制揭示
- 秦晓星觉醒确认——双方都保留了一些
- 穿越者直觉：分身城与重生可能是同一现象的产物

> **设定对齐**：本章涉及的分身城是主角的核心衍生物——位于南方极远处的**活体秘境**，目前为**1层空壳**（仅一层空间），处于混沌孕育中尚未定型降临[参照设定§二]。城内的岩石结构是秘境空间的原始形态，尚未生成怪物或陷阱。

### 场景4-1：尝试连接（500字）
- 第二天，烧退了。虚弱得爬不起来
- 想起那个灰色心核——试着用意识去碰它
- 第一次"跨次元感知"：视野刷地一下切换了
- 不是画面——是深度感知——他站在那座远方的秘境空间里
- 荒芜、空荡、粗糙原始的岩石空间，像被挖空的巨大石室
- **叙事功能**：第一次远程连接——展示感知机制

### 场景4-2：轻松操控（600字）
- 用意念去碰一块碎石——石头滚了
- 轻松自如，像伸手推东西一样自然——不费任何额外力气
- 尝试改变通道走向、调整入口处碎石的堆积方向——同样轻松
- 他坐在宿舍床上，同时让远方的秘境空间里一块石头翻了三次
- **关键设定验证**：日常操控不消耗精神——"像正常思考一样自然"
- **叙事功能**：让读者体验"这事不累"——建立日常操控零消耗的心理模型

### 场景4-3：创造实验（400字）
- 试图"创造"点什么——让石壁长出新的岩石结构
- 面板提示：`[源质不足][当前：0]`
- 再试——让地面上多出一块石头——同样的提示
- 哦。移动已有的东西不费劲。但创造新东西——要资源。
- **叙事功能**：揭示创造行为消耗源质的机制——经营逻辑的基础

### 场景4-4：距离感知（400字）
- 本能地知道自己无法抵达那里——太远了
- 只能感知，无法亲至
- 感应到它在南方极远处——方向模糊，无法精确定位
- 试着往那个方向"走"——意识像撞上一堵墙，越远越模糊
- **叙事功能**：反复强调"分身城遥远、无法抵达"的核心约束

### 场景4-5：面板扫描（400字）
- 下午没课，李冠阳在校园里瞎逛，翻看面板各页
- 发现了一个之前没注意到的功能——右上角有个不显眼的扫描图标
- 试着点击——面板弹出提示：[请选择扫描目标]
- 他抬头，对面正好走来一个同学——下意识点了"扫描"
- 面板弹出：
  ```text
  [扫描结果]
  目标：普通人类
  职业：无
  ```
- 他愣了一下——原来面板能扫出别人是不是职业者
- 又试了一次：一个在操场上跑步的陌生人——同样显示"普通人类，职业：无"
- 内心默默记下：**这个功能可以用来判断对方是不是觉醒者**
- 但面板没有告诉他扫描范围——能扫多远？视线内？感范围内？不知道
- **叙事功能**：面板扫描功能引入——为主角后续判断入侵者是否为觉醒者提供工具，同时保留信息模糊（扫描范围/边界不透明）

### 场景4-7：秦晓星确认（400字）
- 秦晓星发来一张截图——面板：[剑士]
- 确认了。两人都觉醒了。
- 李冠阳没发回自己的双面板截图——只说了"法师"
- 秦晓星也没说她剑士的具体技能——"还在摸索"
- **关系进展**：双方都保留了一些——有限信任
- **叙事功能**：感情拉扯——不是毫无保留，是"信任但不全信"

### 场景4-8：章末（300字）
- 高烧期间那些记忆碎片又闪过一次
- 孤儿院、游戏、被"大运"送走——越来越清晰
- 隐约觉得：那座远方的城，与自己在蓝星重生，是同一时间发生的事
- 分身城……是不是自己重生时诞生的"另一个自我"？
- 说不清。只是一个直觉。
- **章末悬念**：穿越者身份持续暗示——分身城与重生有关联但不是证据

### 第4章信息释放检查

| 释放了 | 没有释放 |
|--------|---------|
| 日常操控不耗精神（移动碎石、调整通道） | 不让主角知道这种能力的上限 |
| 创造行为消耗源质（源质：0） | 不展示源质的完整获取体系 |
| 城在南方极远处、无法精确定位、无法抵达 | 不让主角尝试其他方向或手段 |
| 分身城为1层原始空间（岩石、空荡） | 不让第2层及以上的信息提前暴露 |
| 前世记忆碎片持续回归（孤儿院、游戏、"被送走"） | 不揭示完整穿越机制 |
| 面板扫描功能引入（判断是否职业者） | 不揭示扫描范围/边界 |
| 秦晓星确认剑士+双方都保留信息 | 不让双方关系过度亲密 |

## 第5章《现世·空城》——场景序列

### 核心目标
- 分身城入口在南方某处自动裂开（被动触发）
- 主角尝试实地感知——只能感到"很模糊"
- 远程"看"到自己的城——**1层空壳**，源质为0
- 第一波非觉醒者普通人入侵——核心感知预警启动
- 简单预设驱赶——不露脸——入侵者自动迷路退出
- 源质循环确认——微量增长
- 穿越者暗示延续

> **设定对齐**：本章分身城在混沌中发育到节点后自动产生入口（现世），但尚未完成**定型降临**[参照设定§2.2.2]——入口是暗缝形式，尚未固化为永久性入口。城目前仅有1层空间，无怪物、无陷阱。

### 场景5-1：暗缝（500字）
- 高烧退后第四天晚上——灵魂深处传来一阵震动
- 不是地震，是"有东西裂开了"的体感
- 闭眼感知：一道暗缝在南方某处的荒山中裂开
- 不是他主动打开的——是城自己完成了某个阶段
- 面板没有弹窗——但他就是知道：入口出现了
- **叙事功能**：现世是被动事件——城自我发育到某个节点后自动产生入口

### 场景5-2：感知尝试（500字）
- 周末去郊外，找了一个安静的方向
- 闭眼，集中精神往南方感知——只能感到"在很远的地方，方向模糊"
- 心里默默定位：在西边偏南，但说不清多远、说不清具体在哪
- 没有任何方式能精确定位
- **关键约束确认**：仅知大概方向，无法抵达，无法锁定坐标
- **叙事功能**：让读者和主角一起接受"这座城不在我能管到的地方"

### 场景5-3：空城（600字）
- 回到宿舍，闭眼远程"看"城
- 空的。**1层**。什么都没有。
- 空间约半个足球场——石壁粗糙、地面不平、空气干燥，像被挖空的原始石室
- 没有魔物、没有陷阱、没有任何生命迹象
- 面板显示：`[源质：0]`
- 他试着检查石壁质地、空间的物理边界——秘境有自己的空间规律
- **叙事功能**：展示起点——从零开始不是修辞，是真的零

### 场景5-4：预警+扫描判定（500字）
- 核心感知到入口处的脚步声——三双脚
- **时间差**：他比入侵者更早知道有人进来了
- 闭眼，集中精神，用意念激活面板的扫描功能——对准入口方向
- 面板弹出：[扫描结果] 目标：普通人类（×3） 职业：无
- 确认了——来的是普通人，不是觉醒者
- 三人试探着走进了暗缝，发出惊叹声
- **叙事功能**：核心感知预警的价值建立 + 面板扫描第4章引入后首次实战应用——判定方式从"脚步声猜测"升级为"直接扫描确认"

### 场景5-5：第一次预设驱赶（600字）
- 抉择：现身？不现身？阻止？不阻止？
- **决策**：不露脸——让城自己"劝退"他们
- 利用已有条件进行预设：
  - 通道走向不动（日常可调整，此场景已提前预设好）
  - 在岔路方向提前堆积碎石（利用已有碎石堆）
  - 黑暗环境制造方向错乱感
- **关键**：这些预设操作在入侵者进入前已完成；入侵者进入后主角仅监控不修改——不消耗任何额外精神
- 三人POV：进来→刚才还有路→怎么走不通了→黑得看不到方向→恐惧→退出
- 三人退出后，主角查看面板——`[源质：+1]`
- 入侵者的困惑+体力消耗转化成了源质——**资源循环确认**
- **叙事功能**：
  1. 首次经营验证成功——城底层逻辑成立
  2. 日常预设不耗精神+入侵消耗转化源质——机制闭环
  3. 信息差最大化：读者知道有主，入侵者不知道

### 场景5-6：章末（300字）
- 三人退出后，入口恢复安静
- 李冠阳坐在宿舍床上，看着空荡荡的面板
- 两件事越来越清楚：
  1. 只有他能操控这座城——面板上没有第二个人
  2. 它太远了——他暂时什么都做不了，只能等
- 脑海中闪过一个念头：如果下次来的不是普通人呢？
- **章末悬念**：这座城一定会吸引更多人——他挡不挡得住？在不能亲临的情况下？

### 第5章信息释放检查

| 释放了 | 没有释放 |
|--------|---------|
| 现世是被动触发（暗缝自行裂开） | 不让主角知道完整现世机制 |
| 核心感知预警功能（比入侵者先知道） | 不展示感知范围边界 |
| 日常预设不耗精神（入侵前调好，入侵后只监控） | 不让主角尝试高消耗操作 |
| 入侵者消耗→源质转化（+1） | 不让主角知道转化效率公式 |
| 城只有主角能操控（独有权限） | 不让主角尝试分享权限 |
| 城在南方远处、无法抵达 | 不再重复——已提到为止 |
| 分身城为1层原始空间 | 不让第2层信息提前暴露 |
| 穿越者身份碎片（仅限"不属于这里"的体感） | 不在本章揭示完整前世记忆 |

## 附：地图认知线索链

这是第2-3章关于"地图是什么"的完整认知弧线：

```
发现地图（2-1）
  ↓ 困惑——"这是什么的地图？光点是什么？"
第一次验证（2-4）：光点移动方向=骷髅逼近方向
  ↓ 猜测——"难道这张图就是这座地下城的？"
第二次验证（2-4）：点击光点弹出骷髅信息
  ↓ 初步确认——"地图上的光点=魔物位置"
第三次验证（2-5）：地图整体结构=当前地下城的七层结构+出口标志
  ↓ 完全确认——"这就是这座城的地图"
最终运用验证（3-3）：按地图指示找到真实出口
  ↓ 彻底相信——"地图信息是可靠的"
```

| 阶段 | 章节 | 认知状态 | 关键动作 |
|:----:|:----:|:--------:|:--------:|
| 发现 | 2-1 | 不知道是什么 | 打开地图，困惑 |
| 猜测 | 2-4 | 可能有关系 | 光点移动≈骷髅出现方向 |
| 确认 | 2-4 | 确实是这里的地图 | 点击光点弹出怪物信息 |
| 运用 | 2-5 | 能用来找出口 | 地图上看到裂隙出口标记 |
| 验证 | 3-3 | 彻底信任 | 找到真实出口 |
