# 第一卷《觉醒之始》卷纲

> **卷Slogan**：你以为这是一次意外——其实你的脚已经踩在了棋盘上。
>
> **卷定位**：信息释放期——读者比主角知道得少，主角比入侵者知道得多。
>
> **全书计划**：300万-500万字，1500-2500章。第一卷约110章，为全书奠基期。
>
> **本书核心公式**：信息差+经营成就感+主角前世游戏经验vs懵懂意识+反客为主+双线联动+感情拉扯

## 二、前期·觉醒与扎根（第1-35章·细纲一~三）

> **时期定位**：从零开始，拼命扎根。
> **城阶层数轨迹**：1阶层空壳 → 3阶层
> **核心矛盾**：主角必须让城"活起来"，否则核心暴露，生存无保障
> **核心情绪**：惊恐 → 好奇 → 焦虑 → 掌控 → 宽松

### 细纲一：觉醒与苏醒（第1-10章·2个梗概）

> **爽点闭环**：梦中铺垫+空面板 → 跟入→空间置换→觉醒+前世记忆回归→分身同步苏醒 → 逃生→发现心核与直觉认知 → 现世 → 首波防守萌芽（5-7）→ 冷却与双线推进（8-10）
>
> **情绪曲线**：铺垫（1）→高潮战斗（2）→逃生调整（2末-3）→发现与现世（3-4）→首波防守（5-7）→冷却推进（8-10）
>
> **核心公式渗透**：信息差（梦中碎片+三周铺垫+空面板）→ 双线联动（第2章起法师+城主双面板+前世记忆回归）
>
> **关键设定**：①本卷主角无法亲临分身城，所有经营动作通过分身视角实现；主体对分身的感知始终模糊（仅知大概方向+遥远距离，无法精确定位）。②**日常操控**（查看状态、移动碎石、调整通道方向、制造简单环境变化等）不消耗额外精力，像正常思考一样自然；**消耗资源的是"创造"行为**——创造新魔物、扩建新阶层需要消耗源质。只有**高强度多线战斗操控**（同时操控多阶层防守、应对战术入侵者）才会造成精神疲劳。③主角是穿越者（前世21世纪地球青年，孤儿），觉醒时前世记忆碎片回归。他猜测分身城与自己的重生是同一现象的产物——不是证据，只是直觉。

| **前期·觉醒与扎根** | 1-35 | 1→3阶层 | 0→100+转正 | 手动→半自动预设 | 秦晓星有限信任+女主B入场 | 养鱼策略验证、源质循环确立、穿越者身份确认 |

### 5.4 本卷核心设定约束执行清单

| 约束项 | 执行范围 | 执行方式 |
|--------|:--------:|---------|
| 不出现"地下城"一词（前期） | 第1-20章 | 正文中以"门""异空间""城""核心""入口"等替代 |
| 找不到其他地下城 | 第1-20章 | 第16章两人确认：周围没有城 |
| 入侵者主动找上门 | 全卷 | 没有人被"分配"过来，全靠源质波动吸引 |
| 入侵者无对比基准 | 全卷 | 只觉"诡异"，不认为有人在操控 |
| **分身城遥远，主角无法亲抵** | **全卷** | **主体仅知大概方向（南方极远处），无法精确定位；全卷反复提示主体感知模糊** |
| **日常操控不耗精神** | **全卷** | **移动碎石/调整通道/简单环境变化 = 如思考般自然；仅高强度战斗操控造成精神疲劳** |
| 不告白 | 全卷 | 感情线只有外部事件推动 |
| **分身与本体一体两面** | **全卷** | **分身从始至终是主角的另一半（觉醒时同步苏醒），不是"绑定"来的。"天生就知道"分身城的存在。** |
| **穿越者身份** | **全卷** | **前世21世纪地球青年，孤儿，被"大运"送走。觉醒时前世记忆碎片回归。对分身城的猜测是直觉而非证据。** |
| 被动等待叙事禁制 | 全卷 | 现实线每细纲内必须有主动推进（冲突/事件/线索） |
| 纯种田流水账禁制 | 全卷 | 每细纲必有爽点闭环事件 |
| 降智反派禁制 | 全卷 | 第27-28章"聪明入侵者"已设 |
| 数值杀穿禁制 | 全卷 | 战力靠策略，不靠等级碾压 |

### 5.6 样例书对照表（来源：《异世界，开局变成一座地下城》）

| 样例书元素 | 本项目对应 | 适用章节 | 差异说明 |
|-----------|:---------:|:--------:|---------|
| **地脉之力**（核心资源，自然恢复+1/天） | **源质** | 全卷 | 源质积累方式更多样（受伤转化+自然恢复+入侵消耗转化） |
| **创造新阶层**（每1-2章一个） | 扩张城阶层数 | 第15章起 | 节奏略慢，每梗概1-2阶层，避免升级过快 |
| **魔物创造/改造**（金色天赋） | 自定义魔物生成+行为模式调节 | 第5章起 | 主角的操控权限更大（人脑随时介入），且能调节"演技"，粪怪阴险设计更丰富 |
| **陷阱伪装**（曼德拉草伪装魔药、宝箱怪伪装宝箱） | 环境操控+伪装障碍+诱导型布局 | 第6章起 | 按"看起来有好东西→其实是陷阱"思路，延伸至"演技防守"和"粪怪恶心站位" |
| **分阶层防守**（每阶层不同主题+难度递进） | 第3阶层起分阶层 | 第39章起 | 最终8阶层，每阶层主题不同，含阶层主设计+粪怪配置 |
| **资源循环闭环**（击杀→吞噬→资源→升级） | **受伤转化模式** | 第21章起 | 本项目核心差异：不致死，让入侵者带伤退出，隐藏城有主的事实 |
| **经营波浪节奏**（升级→等待→战斗→反馈→再升级） | 全卷结构 | 全卷 | 完全采用，每5章一个经营小闭环 |
| **"被动等待"叙事**（入侵者主动上门） | 入侵者主动找上门 | 全卷 | 约束条件一致，主角从不主动出击 |
| **外部势力介入**（领主/冒险者公会） | 觉醒者社会→多方 | 细纲七起 | 本项目外部势力后置，前期专注于城与校园双线 |
| **信息激发方式**（第1章面板系统直接展示全貌） | "天生就知道"的直觉认知 | 第3-4章 | 主角对分身城的信息不是通过系统文字获得的，而是"天生就知道"，类似本能的认知 |
