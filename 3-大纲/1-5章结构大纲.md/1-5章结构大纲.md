# 1-5章结构大纲（新版）

> **黄金前5章** 三拨人同时吸入 → 被迫组队 → 双线并行
>
> 基于以下约束条件：信息混沌、前20章找不到其他地下城、入侵者主动找上门、法师线以现实活动为主、核心经营是唯一战场。

---

## 第1章《门》——场景序列

### 核心目标
- 建立李冠阳的日常生活基线
- 引入秦晓星：因被追撞入废弃商场
- 石门自开——三拨人同时被吸入地下城
- 章末惊悚：有东西过来了

### 场景1-1：碎片（300字）
- 李冠阳梦中惊醒。核心面板碎片闪现——黑暗、石壁、源质条
- 不是第一次做这个梦。手心全是冷汗
- **叙事功能**：开局信息碎片，制造悬念——"他身上有什么东西"

### 场景1-2：日常（500字）
- 白天上课。舍友在聊"学校后面那片旧商业区封了——说是地基异常"
- 主角听了就过。手机刷到朋友圈有人发"真有人消失了"，评论区半信半疑
- **信息混沌**：没有官方说法，没有公开报道。普通人还在"到底真的假的"阶段

### 场景1-3：走廊（300字）
- 下课走廊里，有女生从旁边经过
- 李冠阳看了一眼——不熟，但见过。秦晓星。同校，不同系
- 他不知道她的名字，但脸有印象
- **叙事功能**：伏笔——"这个人在后面会再出现"

### 场景1-4：目睹（600字）
- 下午回宿舍，路过旧商业区外围
- 看到秦晓星往废弃商场方向跑——脚步急，回头看了一眼，脸色不对
- 后面三个人跟过来了
- 李冠阳停住了。他在犹豫
- **叙事功能**：第一转角——主角面临"管不管"的选择

### 场景1-5：石门自开（600字）
- 秦晓星跑到废弃商场地下入口附近——无路可走。追兵也到了
- 然后石门自己开了。不是任何人推的——门轴无声转开，露出暗光通道
- 所有人都愣住了。李冠阳在后面看到这一幕
- **叙事功能**：世界观展示——地下城的门不是人开的，是城自己开的

### 场景1-6：吸入（400字）
- 视野切换。不是吸力——是"区域置换"
- 李冠阳脚下变成石地。秦晓星在几米外。追兵三人也在
- 石门在身后合上。地下洞穴空间。符文灯
- 所有人都进来了
- **关键信息**：三拨人同时吸入，不是只有主角一个人

### 场景1-7：金手指登场（400字）
- 核心面板第二次弹出——比开场清晰。源质条闪烁
- 右下角新文字：「检测到异空间冲突。」
- 黑暗中传来声音。震感。有东西过来了
- **叙事功能**：金手指的第一次正式登场，但主角没时间研究

### 场景1-8：章末钩子（300字）
- 追兵头目开口："这什么地方？"
- 没人回答他
- 震感越来越近。不是人的脚步声。是东西。一群
- **章末悬念**：战斗即将开始，但没有人准备好

### 第1章信息释放检查

| 释放了 | 没有释放 |
|--------|---------|
| 世界背景：超凡事件发生约3周，普通人半信半疑 | 不解释"地下城"是什么 |
| 主角现状：普通大学生，有梦中碎片 | 不展示完整面板 |
| 女主存在（秦晓星，同校不同系） | 不揭示女主觉醒潜力 |
| 石门自开机制（城主动开门） | 不让主角理解"为什么门会开" |
| 三拨人同时被吸入 | 不让任何人知道"怎么回去" |

## 第2章《觉醒》——场景序列

### 核心目标
- 怪物群冲击——混乱战斗
- 李冠阳被逼觉醒（法师）
- 秦晓星被逼觉醒（战斗型职业）
- 三拨人分散——男女主被迫一起走
- 找到出口，逃出

### 场景2-1：混乱（500字）
- 怪物群从通道冲出
- 三拨人各自反应：追兵打手1冲上去被击退；秦晓星后退找武器；李冠阳本能闪避
- 全员被迫加入战斗——没有退路
- **叙事功能**：展示地下城的第一波战斗——混乱、无组织、所有人都没准备好

### 场景2-2：李冠阳觉醒（600字）
- 李冠阳被战斗逼到边缘
- 面板弹出：`[职业引导完成：法师]`
- 身体变化——源质流入四肢
- 第一次放技能——火球术（本能释放，不熟练）
- **觉醒方式**：战斗中觉醒，不是进入就觉醒

### 场景2-3：秦晓星觉醒（500字）
- 她在另一侧被怪物逼近
- 逼到死角——本能爆发
- 面板弹出：职业确立（战斗型/具体职业待定）
- **对比**：她的觉醒更"硬"——不像李冠阳有系统引导感，更像本能激活

### 场景2-4：分散（400字）
- 震动/坍塌/地下城机制触发
- 三拨人在混乱中被隔开
- 追兵三人被隔到另一条通道
- 秦晓星和李冠阳被迫退到同一侧通道
- **叙事功能**：第一次将男女主锁在一起——不是浪漫安排，是生存压力

### 场景2-5：寻找出口（500字）
- 男女主零交流走了一段
- 两人浑身是伤，沉默前行
- 秦晓星先开口："你也是被卷进来的？"
- 简短交换信息——不认识，但在学校见过
- **关系起点**：零信任，被迫同行

### 场景2-6：找到出口（400字）
- 找到裂缝/出口——不是正门，是城墙裂缝
- 两人逃出去
- 回到废弃商场。石门消失
- 追兵还没出来
- **叙事功能**：出口消失意味着地下城进入冷却期

### 场景2-7：章末（300字）
- 两人站在废弃商场外。浑身是伤
- 秦晓星："你那个……面板还在吗？"
- 李冠阳："在。"
- 对视。都知道对方不是普通人了
- **章末基调节器**：危机的短暂结束，但两人都知道这只是开始

### 第2章信息释放检查

| 释放了 | 没有释放 |
|--------|---------|
| 觉醒过程（战前觉醒触发） | 不解释"地下城"这个名称 |
| 李冠阳的法师职业 | 不展示完整技能树 |
| 秦晓星觉醒（战斗型） | 不揭示她的完整职业 |
| 三拨人分散（追兵隔开） | 不让男女主安全——他们只是逃出来了 |
| 出口消失机制 | 不让任何人知道下次入口在哪 |

## 第3章《双面板》——场景序列

### 核心目标
- 后遗症：李冠阳发现灰色「心核」
- 第一次连接——感知到另一座城
- 秦晓星开始独立适应觉醒者身份
- 两人保持联络
- 章末提示：降临倒计时开始

### 场景3-1：后遗症（500字）
- 李冠阳回宿舍。浑身痛
- 查看面板——发现多了一个灰色的「心核」
- 心核下方一行小字：「已绑定[未命名地下城]」
- **叙事功能**：暗示他除了法师身份之外还有另一个维度的能力

### 场景3-2：试探（500字）
- 用意念点开心核
- 第一次加载缓慢——像是系统在同步什么
- 加载完成后，意识中出现**第二个视角**
- 不是画面——是深度感知——"在地下城深处，有一个核心在等待他"
- **叙事功能**：让读者和主角同时探索双面板的边界

### 场景3-3：第一次连接（600字）
- 晚上深度连接心核
- 感知到地下城内部——空间大小、墙壁质地、空气中的能量
- 用意念移动某块石头——**石头动了**
- 吓了一跳，断开连接
- **叙事功能**：展示核心分身的能力——不是第二人格，是"第三只手"

### 场景3-4：秦晓星线（400字）
- 她在调整。查到自己的职业信息
- 发现自己觉醒了某个不常见的战斗职业
- 两人保持联络——交换了联系方式
- **叙事功能**：独立线建立——秦晓星有自己的觉醒者路径

### 场景3-5：双线并行（500字）
- 李冠阳开始理解双面板运作方式
- 明面：法师。暗面：城主
- 发现可以同时做两件事——表面上听课，意识一部分沉浸在地下城中
- **代价**：精神消耗加快——第三节课开始头痛
- **叙事功能**：展示双面板的代价机制

### 场景3-6：章末（300字）
- 核心面板推送第一条提示：
- `[降临倒计时开始]`
- 主角不知道这意味着什么
- **章末悬念**："降临"是什么？好的还是坏的？

### 第3章信息释放检查

| 释放了 | 没有释放 |
|--------|---------|
| 双面板存在（法师+心核） | 不展示全部操控权限 |
| 核心分身能力雏形（感知+移动物体） | 不让主角完全理解"分身是什么" |
| 精神消耗机制 | 不让主角知道消耗极限在哪 |
| 秦晓星独立线建立 | 不揭示她的完整职业信息 |
| 降临倒计时 | 不让主角知道降临意味着什么 |

## 第4章《降临》——场景序列

### 核心目标
- 72小时倒计时——主角试图手动干预失败
- 探索自己的城——空的，什么都没有
- 秦晓星在适应觉醒者身份——两人校园碰面
- 降临发生——入口在郊外树林
- 第一次亲手操控入口形状

### 场景4-1：倒计时（500字）
- 72小时倒计时
- 李冠阳不知道这意味着什么
- 试着手动干预降临——失败
- 面板只有一行模糊描述："降临期——地下城与现实世界联通。外人可进入。"
- **叙事功能**：降临不可控——主角被动接受

### 场景4-2：探索自己的城（600字）
- 课余时间通过核心面板感知自己的城
- 空的。1层。什么都没有
- 面积约半个足球场
- 需要源质来造东西——源质为0
- **叙事功能**：源质体系基础概念——"需要先运营才有产出"

### 场景4-3：秦晓星线（400字）
- 她也在适应自己的觉醒
- 两人在校园里碰面
- 对话简短——都在消化"世界变了"
- **关系进展**：从零信任过渡到"可以说话的人"

### 场景4-4：降临发生（500字）
- 72小时后。地下城降临
- 李冠阳在宿舍里感受到一阵震动——来自灵魂深处
- 闭眼感知：入口出现在郊外树林
- 面板弹出：`[降临期：7天]`
- **叙事功能**：降临是有期限的——不是永久开放

### 场景4-5：第一次操控（600字）
- 逃课去郊外。找到入口——地面上黑漆漆的洞口
- 用法师身份进入。亲手摸到自己的城
- 用核心操控调整入口形状——洞口缩小
- 操控10分钟就开始头痛
- **叙事功能**：展示核心操控的实际应用

### 场景4-6：章末（300字）
- 意识到两件事：
  1. 自己的城能操控——和别人不一样
  2. 这个能力不能让人知道
- **章末悬念**：他的城和别人不一样——这是优势，也是危险

### 第4章信息释放检查

| 释放了 | 没有释放 |
|--------|---------|
| 降临机制（72小时倒计时+7天窗口） | 不展示完整的降临规则 |
| 核心操控基础（移动物体、调整入口） | 不让主角知道所有操控项 |
| 源质基础概念（需要经营获取） | 不让主角知道如何高效获取源质 |
| 秦晓星与主角的关系进展 | 不让两人关系过度亲密 |
| 主角意识到自己与众不同 | 不让主角完全理解差异的原因 |

## 第5章《第一波入侵者》——场景序列

### 核心目标
- 降临第3天——三个陌生人进入
- 主角抉择：露脸还是隐藏？
- 用环境驱赶——制造雾气，移动碎石墙
- 秦晓星独立事件——她的第一次战斗
- 章末反思：需要自动化防守体系

### 场景5-1：异动（500字）
- 降临第3天。李冠阳在地下城内层布设基础障碍
- 核心分身感知到入口处有三个人的脚步——不是他的
- 时间差：主角比入侵者更早知道他们进来了
- **叙事功能**：核心感知的预警价值——比入侵者快一步

### 场景5-2：抉择（600字）
- 三个人已经走进第一层
- 主角面临选择：现身？不现身？阻止？
- **第一个战术决策**：不露脸，用环境驱赶
  - 调整通道墙壁——把路引向难走的方向
  - 制造雾气——降低能见度
  - 移动碎石堆——制造障碍
- **叙事功能**：主角第一次在"自己的城"中做出战术决策

### 场景5-3：三人视角（500字）
- 切换到入侵者POV：
  - "这个地方好奇怪，上次进的异空间不是这样"
  - "没有看到怪物……但是好像一直在变？"
  - "刚才那边还有路，现在变成墙了？"
- 不是觉醒者——只是三个好奇的普通人
- 找不到路→恐惧→退出
- **关键**：没有对比基准——只觉得"诡异"，不知道有人在操控

### 场景5-4：秦晓星独立线（400字）
- 她在第5章遇到自己的第一个小型战斗/事件
- 独立线——与李冠阳不同步
- **叙事功能**：展示秦晓星的独立成长路径

### 场景5-5：反思（500字）
- 三人退出后，主角断开核心感知——精神力几乎耗尽
- 意识到：这次是普通人，下次可能是觉醒者
- 不能每次都硬扛——需要"自动化防守体系"
- 用源质生成第一批基础怪物
- 规划陷阱布局
- **叙事功能**：从防守到经营的转折——主角开始认真经营

### 场景5-6：章末（300字）
- 核心面板提示：`[降临期剩余：2天]`
- 他要在2天内让城至少能拦住第二波人
- **章末悬念**：冷却期后——谁能进来？进多少人？他挡不挡得住？

### 第5章信息释放检查

| 释放了 | 没有释放 |
|--------|---------|
| 降临期有外人进入 | 不让入侵者意识到有人在操控 |
| 核心感知预警功能 | 不展示完整感知范围 |
| 基础操控有效性（迷雾、移墙） | 不让主角消耗过量源质 |
| 秦晓星独立事件 | 不揭示她的战斗结果 |
| 自动化防守需求 | 不给"完美的防守方案" |

## 附：新版结构优势对比

| 对比项 | 旧版（单人线） | 新版（三拨人+双线） |
|:------:|:-------------:|:------------------:|
| 开局复杂度 | 单人觉醒→探地图 | 三拨人同时吸入→混乱→分散 |
| 女主引入 | 无（待定） | 第1章出现，第2章觉醒 |
| 感情线起点 | 无 | 零信任→被迫同行 |

| 双线并行 | 无 | 第3章起男女主各自独立线 |
| 金手指揭示节奏 | 单人摸索 | 心核贯穿 + 战斗觉醒 |
